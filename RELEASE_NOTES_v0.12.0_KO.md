# v0.12.0 지도 성·지명 가로쓰기 및 동적 라벨

Steam 일본어판의 세로형 지도 표기를 영문판 계열의 가로형 성·지명 라벨로 전환했습니다.

- 성 이름과 지명을 가로로 표시합니다.
- 일반 라벨 배경, 소유 세력 색상, 선택 시 금색 하이라이트, 아이콘과 앵커 위치를 함께 전환합니다.
- 라벨 폭을 고정 글자 수가 아니라 실제 UTF-16 문자열 길이로 계산합니다.
- 공백과 `성` 접미사를 포함하며 3자·5자 같은 최대 글자 수 제한을 두지 않습니다.
- 영문판 배너 자산은 사용자 게임의 언어 리소스에서 파생한 압축 오버레이로 설치하며, 원본 리소스 아카이브는 수정하지 않습니다.

## 설치기 변경

정적 EXE 마스터 설치기 레지스트리를 v2로 확장하고 `004` 패치를 추가했습니다.

- `001~003`이 이미 적용된 EXE에서는 `004`만 자동 적용합니다.
- 원본 EXE에서는 `001~004`를 순서대로 적용합니다.
- 확장된 PE와 추가 페이로드까지 해시로 검증합니다.
- 일부만 적용되었거나 등록 범위 밖의 바이트가 있으면 교체하지 않고 중단합니다.
- 재실행 시 모든 항목을 `Applied`로 판정하고 파일을 다시 쓰지 않습니다.
- `RESTORE_ORIGINAL_NOBU16PK_EXE.bat`로 원본 복원이 가능합니다.

게임 실행 파일 자체는 배포 ZIP에 포함되지 않습니다.

- 실게임 해상도: `1920x1080`

- 일반 상태와 선택 하이라이트 상태에서 서로 다른 길이의 한국어 성 라벨 확인
- 392개 성 이름 데이터의 최장 표시 문자열 9자를 정적 폭 모델로 확인
