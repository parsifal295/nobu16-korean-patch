# NOBU16 PK 한글 패치 v0.15.1 릴리즈 노트

## PORT3 순정 프로필 긴급 수정

v0.15.1은 v0.15.0의 `Resource stage Preflight failed` 문제를 수정한
핫픽스입니다. 번역, DLC 지원, 이미지 검수본과 성 이름 세로쓰기·가로쓰기
선택 기능은 v0.15.0과 동일합니다.

### 수정 내용

- v0.15.0은 `RES_JP_PK_PORT/res_lang_pk_port3.bin`의 이미지 작업 중간본을
  Steam JP 1.1.7 순정 원본으로 잘못 등록했습니다.
- 그 결과 실제 순정 파일이 `unknown`으로 판정되어, 리소스 적용 전에 패처가
  안전하게 중단됐습니다.
- v0.15.1은 올바른 순정 파일을 입력으로 고정하고 PORT3 BSDIFF40 델타를
  다시 생성했습니다.

```text
순정 PORT3
크기: 43,460,485 bytes
SHA-256: BE1361E17341D433931EB5740B228EF1842BF6DF2F01D4F582CE790A9A57A154

v0.15.1 목표 PORT3
크기: 43,161,969 bytes
SHA-256: BA739C28A8EE1A47C8085339F98FDCF4F317302316F93C3F74E413DB2AFEADC9
```

v0.15.0 오류 보고에 `writes_performed: false`가 표시됐다면 해당 실행에서는
게임 파일이 변경되지 않았습니다. Steam 게임 파일 무결성 확인을 완료한 뒤
v0.15.1 ZIP을 게임 폴더에 풀고 `APPLY_KOREAN_PATCH.bat`를 실행하십시오.

v0.15.0이 이미 정상 적용되어 필수 리소스가 목표 상태라면 v0.15.1은 해당
리소스를 다시 쓰지 않습니다. 통합 설치기를 다시 실행해 원하는 성 이름 표시
방식 `1` 세로쓰기 또는 `2` 가로쓰기를 선택할 수 있습니다.

### DLC 처리

- Base·PK DLC 번역 프로필 105개를 계속 지원합니다.
- 보유하지 않은 DLC 파일은 `missing_optional`로 건너뜁니다.
- DLC가 하나도 없어도 필수 리소스 16개는 정상적으로 설치됩니다.
- 패치 후 DLC를 추가했다면 통합 설치기를 다시 실행하여 새로 설치된 순정 DLC만
  번역할 수 있습니다.

### 검증 결과

- 올바른 순정 PORT3가 포함된 121개 전체 순정 벡터의 프로필 생성 통과
- 111개 BSDIFF40 델타의 목표 재구성 검증 통과
- 공개 EXE에서 DLC 0개 상태 사전검사 및 필수 리소스 16개 적용 통과
- 적용 후 PORT3 목표 SHA-256 `BA739C28…ADC9` 확인
- DLC 1개 후속 설치·번역 통과
- 필수 리소스와 추가 DLC 복원 통과
- 복원 후 PORT3 순정 SHA-256 `BE1361E1…A154` 확인
- 동일 입력 반복 빌드의 ZIP 바이트·SHA-256 일치
