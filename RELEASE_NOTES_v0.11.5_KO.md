## v0.11.5 — 가공 히메 한글 작명 허용

### 수정

- 가공 히메 작명 화면에서 한글 이름이나 독음을 직접 입력할 때
  `사용할 수 없는 문자` 오류로 진행할 수 없던 Issue #62를 수정했습니다.
- 공용 문자 검증기를 바꾸지 않고, 가공 히메 작명 함수 안의 네 호출부만
  성공 처리했습니다. 따라서 무장 에디트와 다른 입력 화면의 동작은 바꾸지
  않습니다.
- 기존 정적 EXE 해시
  `2E098ECB5E4335DC264F865306B990B724EA7C242B1B9F87FFC5EE2E7191797C`를
  통합 패치 해시
  `7CA2F1D59E02650C67F343F0776F6D05517C0486B65168E63A9AE4CBCAAFDBB2`로
  갱신합니다.
- 이름·독음 공란 검사, 성+이름 합산 길이 제한, 결정 버튼의 다른 활성화 조건은
  유지됩니다.
- 이전 v0.11.0~v0.11.4 정적 무장 에디트 패치가 적용된 EXE는 원본 백업이
  남아 있으면 가공 히메 작명 수정분만 인플레이스 업그레이드합니다.
- v0.11.4의 정책 효과 퍼센트 단위 복구와 15개 게임 리소스는 변경하지
  않았습니다.

### 검증

- 원본 EXE 신규 설치, 이전 정적 패치 EXE 인플레이스 업그레이드, 최신 패치
  재실행, 원본 복구를 자동 검증했습니다.
- 설치기 계약은 원본·언팩·기존 패치·최종 EXE SHA-256, 아홉 개 패치 전 바이트,
  함수 prefix 및 PE 체크섬을 확인한 뒤에만 EXE를 원자적으로 교체합니다.
- 관련 회귀 테스트 51개를 통과했습니다.
- Steam JP 1.1.7 실게임을 완전히 종료·재실행한 3840×2160 창 모드에서,
  사용자 확인으로 한글 입력과 결정 동작을 검증했습니다.

### 설치

1. Steam/런처의 게임 언어를 **Japanese**로 설정합니다.
2. 게임과 런처를 완전히 종료합니다.
3. ZIP 전체를 게임 폴더에 덮어씁니다.
4. 무장 에디트 또는 가공 히메 작명에 한글을 사용할 경우
   `APPLY_STATIC_OFFICER_EDITOR_FIX.bat`를 한 번 실행합니다.
