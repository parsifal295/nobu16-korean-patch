# 노부나가의 야망 신생 한글 패치 — 메인 메뉴 시험판

이 배포본은 영문판처럼 가로쓰기와 긴 이름을 사용할 수 있도록 간체 중국어 리소스 경로를 이용하는 한글화 작업의 첫 단계입니다. 현재 번역 범위는 메인 메뉴이며, 간체 중국어 리소스의 메시지와 한글 글꼴만 파일 방식으로 재구성합니다.

## 설치

1. 압축을 푼 배포 폴더를 게임 설치 폴더 바로 아래에 놓습니다. 배포 폴더와 `MSG_PK`, `RES_SC` 폴더가 같은 상위 폴더에 보여야 합니다.
2. 게임과 공식 런처를 모두 종료합니다.
3. `VERIFY_PACKAGE.bat`를 실행해 배포 파일이 온전한지 확인합니다.
4. `APPLY_KOREAN_PATCH.bat`를 실행합니다.
5. 공식 런처를 직접 열고 언어를 **Simplified Chinese(간체 중국어)** 로 선택한 뒤 게임을 시작합니다.

적용 도구가 게임이나 공식 런처를 대신 실행하지 않으며, 언어 설정도 변경하지 않습니다.

## 원상 복구

게임과 공식 런처를 종료한 뒤 `RESTORE_ORIGINALS.bat`를 실행합니다. 설치 때 만든 검증된 원본 백업으로 메시지와 폰트 리소스를 함께 복구합니다.

## 안전장치

- 지원하는 원본 두 파일의 크기와 SHA-256이 모두 일치해야 설치됩니다.
- 결과 파일도 미리 고정한 크기와 SHA-256으로 검증한 뒤 교체합니다.
- 두 파일 사이에서 작업이 중단되면 거래 기록을 이용해 다음 적용 또는 복구 때 원본 상태로 수습합니다.
- 게임 실행 파일, 공식 런처, 레지스트리는 읽거나 변경하지 않습니다.
- 실행 중인 게임 또는 런처가 감지되면 작업을 거부합니다.
- 최종 사용자는 Python, Ghidra, 별도 글꼴 설치가 필요하지 않습니다.

지원하지 않는 버전이라는 오류가 나오면 게임 파일을 검증해 원본으로 되돌린 뒤 다시 시도하십시오. 해시가 다른 파일에 억지로 적용하지 않습니다.
