# 파일 전용 배포 정책

이 패치는 실행 중 프로그램의 메모리에 접근하지 않고 디스크의 메시지 및 폰트 리소스만 오프라인으로 재구성합니다.

배포물에는 상용 원본 파일 전체가 들어 있지 않습니다. 자체 번역문, 공개 글꼴에서 만든 추가 글리프 자료, 구조 레시피, 해시, 공개 소스 적용기와 라이선스만 포함합니다.

설치기는 다음 원칙을 지킵니다.

1. 게임과 공식 런처가 종료된 상태에서만 동작합니다.
2. 지원 원본의 크기와 SHA-256을 먼저 확인합니다.
3. 결과를 별도 파일로 완성하고 목표 SHA-256을 확인한 뒤 같은 볼륨에서 교체합니다.
4. 원본 백업과 거래 기록을 게임 폴더에 보관합니다.
5. 중단된 거래를 발견하면 검증된 원본 두 파일로 일관된 상태를 복구합니다.
6. 게임 실행 파일, 공식 런처, 레지스트리 설정은 변경하지 않습니다.
