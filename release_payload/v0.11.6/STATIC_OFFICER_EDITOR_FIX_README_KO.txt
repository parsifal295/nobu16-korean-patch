NOBU16 PK Steam JP 1.1.7 v0.11.6
패치 레지스트리 기반 마스터 — 영구 정적 EXE 패치

이 ZIP에는 NOBU16PK.exe가 들어 있지 않습니다. 사용자의 Steam JP 1.1.7
원본 EXE를 PC 안에서 한 번 언팩하고, 등록된 패치 항목별 상태를 검사한 뒤
필요한 항목만 적용하여 영구 교체합니다. 이미 v0.11.0~v0.11.4의 무장
에디트 패치 또는 v0.11.5의 성명 검증 패치가 적용된 EXE도 적용된 항목은
건너뛰고 나머지만 추가합니다.
설치 뒤에는 매번 실행할 helper나 메모리 패치가 없습니다.

등록된 패치
- 001: 무장 에디트 한글 성명 검증, 5개 지점
- 002: 가공 히메 한글 작명 검증, 4개 지점
- 003: 한국어 상단 헤더 가로 배치, 12개 지점

마스터 설치기는 각 항목을 Applied 또는 Pending으로 판정합니다. 한 항목 안에
적용 전·후 바이트가 섞인 부분 적용 상태나 알 수 없는 바이트가 있으면 EXE를
교체하지 않고 중단합니다. 000-PatchRegistry.psd1은 포함할 패치 파일과 해시,
모두 적용된 최종 EXE 해시를 고정합니다. 향후 패치는 Patches 폴더에 번호가
붙은 데이터 파일을 추가하고 이 레지스트리에 등록하는 방식으로 분리합니다.

설치 및 업그레이드
1. 게임 언어를 Japanese로 선택합니다.
2. 게임과 런처를 완전히 종료합니다.
3. 이 ZIP의 모든 파일을 게임 폴더에 풀고, 리소스 덮어쓰기를 허용합니다.
   일반적인 폴더 예: ...\SteamLibrary\steamapps\common\NOBU16
4. 게임 폴더 최상단의 APPLY_STATIC_EXE_PATCHES.bat를 실행합니다.
   기존 APPLY_STATIC_OFFICER_EDITOR_FIX.bat도 같은 마스터 설치기를 호출합니다.
5. "Completed"가 표시되면 평소처럼 Steam -> 런처 -> 게임으로 실행합니다.

기존 패치에서 업그레이드할 때는 이전 설치기가 만든 다음 원본 백업이 같은
게임 폴더에 있어야 합니다.

  NOBU16PK.exe.staticfix.original_1.1.7

설치기는 원본 또는 기존 패치 EXE를 모든 등록 지점의 적용 전 상태로
정규화했을 때 검증된 Steamless 원본과 완전히 같은지도 확인합니다. 따라서
등록되지 않은 EXE 변경, 부분 적용, 알 수 없는 바이트가 하나라도 있으면 EXE를
교체하지 않습니다. 원본에서 새로 설치할 때는 위 이름으로 원본 백업을
생성합니다.

실게임 확인
- 2048x1152 해상도에서 게임 프로세스를 완전 종료 후 재실행하여 확인했습니다.
- 한글 군주명이 길어도 명성 칸과 겹치지 않고, 명성·기력·책 수치가 북미판 기준 가로 위치에 표시됩니다.
- 해상도 변경 뒤 재실행하지 않은 화면은 적용 여부 판정에 사용하지 않습니다.

원복
- 게임과 런처를 종료한 뒤 RESTORE_ORIGINAL_NOBU16PK_EXE.bat를 실행합니다.
- 기존 패치 EXE와 최신 패치 EXE 모두 같은 원본 백업으로 복구할 수 있습니다.
- Steam 파일 무결성 검사나 게임 업데이트도 EXE를 원본으로 되돌릴 수
  있습니다. 이후에는 정확히 같은 1.1.7 원본일 때만 다시 적용할 수 있습니다.

무엇이 바뀌나
- 무장 에디트가 기존 한글 성/이름 및 읽기 필드를 사용할 수 없는 문자로
  거부하지 않도록 기존 문자 검사 네 곳을 우회합니다.
- 기존 한글 성명의 합산 6자 제한을 기존 계속 경로로 보냅니다.
- 가공 히메 작명 화면에서 상속된 성/성 독음과 입력한 이름/이름 독음의 문자
  검사 네 곳을 해당 화면 안에서만 성공 처리합니다.
- 가공 히메의 이름·독음 공란 검사, 성+이름 합산 길이 제한 및 결정 버튼의
  나머지 활성화 조건은 그대로 유지됩니다.
- 공통 문자 검증 함수는 수정하지 않으므로 다른 입력 화면에는 영향을 주지
  않습니다.
- 한국어 상단 패널 폭과 이름 표시 폭을 북미판 규격으로 넓힙니다.
- 명성·기력·책 수치와 상호작용 앵커를 북미판의 실제 가로 좌표로 옮겨 긴
  한글 이름이 명성 칸을 침범하지 않도록 합니다.
- 원형 진행 위젯, 시간 배속 버튼, 세로 좌표 및 게임 리소스 파일은 수정하지
  않습니다.

제한
- 무장 에디트와 가공 히메 작명 화면에서는 원래 금지된 문자 종류 검사도 함께
  우회됩니다. 공란 및 길이 검사는 유지됩니다.
- 이것은 한글 IME 입력 기능 자체를 추가하는 패치가 아닙니다.
- 최종 EXE는 Steam 보호 래퍼와 원본 Authenticode 서명이 제거된 NotSigned
  파일입니다. 반드시 Steam 또는 1.1.7 런처에서 게임을 시작하십시오.
- Windows의 .NET Framework 4.5.2 이상이 필요합니다.
