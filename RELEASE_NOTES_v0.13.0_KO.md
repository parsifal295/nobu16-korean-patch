# NOBU16 PK 한글 패치 v0.13.0 릴리즈 노트

> 현재 릴리즈 후보 문서입니다. v0.13.0 자산은 아직 GitHub Release로
> 게시하지 않았습니다.

## v0.13.0 — 번역 품질 개선·고해상도 지도 라벨·명소 가로쓰기

### 번역 품질 개선

- 순정 PC 일본어 원문과 PC EN/SC/TC를 대조해 고확신 항목만 반영했습니다.
- 이벤트 대사 **26건**(Base 8건, PK 18건)과 인물 대사 **51건**
  (Base 13건, PK 38건), 총 **77건**을 보정했습니다.
- 변경 리소스는 다음 네 파일입니다.
  - `MSG/JP/ev_strdata.bin`
  - `MSG/JP/msggame.bin`
  - `MSG_PK/JP/msgev.bin`
  - `MSG_PK/JP/msggame.bin`
- 기존 줄바꿈 수, 리터럴 경계, 불투명 바이트, 제어 마커와 종료 코드를
  보존했습니다. 912px 경계에 걸리거나 실제 화면 판정이 필요한 후보는
  이번 릴리즈에서 제외했습니다.
- 스위치판 한글은 번역 기준으로 사용하지 않았습니다.

### Issue #68 — 4K 지도 라벨 그래픽 깨짐

- v0.12.0의 가로 라벨 패치가 고해상도 지도 분기에서 대응 아틀라스를 함께
  제공하지 않아 4K에서 라벨과 지도 UI 조각이 깨지는 문제를 수정했습니다.
- 일반 해상도용 아틀라스와 함께 고해상도 분기의 `4096×2048` 및
  `4096×4096` 텍스처 구성을 정적 오버레이에 포함했습니다.
- 게임 리소스 아카이브는 수정하지 않으며, 사용자의 검증된 EXE에만
  해시 고정 정적 패치를 적용합니다.

### Issue #70 — 명소 이름 가로쓰기

- 일본어판 분기의 명소 이름이 세로쓰기 컨테이너에 남아 아쓰타 신궁처럼
  여러 글자가 한 점에 뭉쳐 보이던 문제를 수정했습니다.
- 영문판과 같은 가로쓰기 명소 소유자 라벨 생성 경로 11곳만 정확히 바꿨습니다.
- 800×450에서 해상도를 선택한 뒤 `NOBU16PK.exe`를 완전히 종료하고 새
  프로세스로 재실행해 아쓰타 신궁의 가로 표시를 확인했습니다.

### 설치기 005 통합

- **v0.12.0 사용자는 v0.13.0 ZIP을 게임 폴더에 덮어쓴 뒤 반드시 새
  `APPLY_STATIC_EXE_PATCHES.bat`를 다시 실행해야 합니다.** 파일만 덮어쓰면
  EXE에는 `005`가 적용되지 않습니다.
- v0.12.0에서 배포한 `004` 정의와 페이로드는 변경하지 않습니다.
- 새 `005-DualResolutionAndHorizontalLandmarks`에 Issue #68과 Issue #70을
  함께 넣었습니다.
- v0.12.0에서 `001~004`가 이미 적용된 EXE는 `005`만 자동 적용합니다.
- 원본 Steam JP 1.1.7 EXE는 `001~005`를 순서대로 적용합니다.
- v0.13.0 설치기를 다시 실행하면 다섯 항목을 모두 `Applied`로 판정하고
  EXE를 다시 쓰지 않습니다.
- 연속 구조 오버레이를 안전하게 판정하기 위해 마스터 레지스트리를 v3으로
  갱신했습니다. 선행 패치가 없으면 `005`를 대기 상태로 두며, 정규화와 검증은
  구조 패치의 역순으로 수행합니다.

전체 적용 후 정적 EXE SHA-256:

```text
BE983A61C81008289E2483D552122C0BE3299B5F8DD4A557FA14DA2663AC7BD6
```

v0.13.0 ZIP은 아직 빌드·게시하지 않았으므로 이 문서에는 배포 ZIP 해시를
기재하지 않습니다. ZIP에는 `NOBU16PK.exe`가 포함되지 않습니다.
