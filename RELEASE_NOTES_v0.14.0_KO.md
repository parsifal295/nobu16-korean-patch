# NOBU16 PK 한글 패치 v0.14.0 릴리즈 노트

## 이벤트 메시지 가독성

- 이벤트 메시지 전용 글자 크기를 36px에서 30px로, 줄간격을 10에서 8로 조정했습니다.
- 이벤트 내레이션 부모 폭을 882px에서 972px로 넓혀, 실제 텍스트 레인 912px을
  확보했습니다.
- 주·대체 자동줄바꿈 경로의 한도를 전각 20자급에서 30자급으로 조정했습니다.
  수동 개행, 색상 태그, 런타임 토큰은 데이터 그대로 보존합니다.

## 번역·표기 보정

- Base·PK 인물 대사 B17의 17개 레코드, 24개 리터럴에서 이름 경계 공백, 조사,
  문장 연결을 원문 구조와 대조해 보정했습니다.
- PK 이벤트 대사에서 아네가와 전투 14건을 포함한 20건의 수동 줄바꿈을 문맥에 맞게
  재배치했습니다. 문장 축약 없이 색상 태그와 런타임 토큰을 보존했고, 모든 줄은
  30px·유효 폭 912px·최대 4줄 기준을 통과했습니다.
- 무장 성·이름 독음 칸을 한국 한자음으로 전환했습니다. PK `msgdata.bin` 4,569칸과
  Base `strdata.bin` 4,556칸이 대상이며, `직전신장`처럼 공백 없는 한글 독음으로
  표시됩니다. 일반 표시 무장명은 변경하지 않습니다.
- 동적 조합 성명이 붙어 보이던 전용 별칭 성 블록 5건의 성·이름 경계 공백을
  보정했습니다. 재사용 조각은 다른 무장명에 영향을 줄 수 있어 제외했습니다.

## 지도 보조 표시

- 임전 숫자를 이동된 상태 아이콘 바로 오른쪽에 정렬했습니다.
- 성주 미임명 경고 아이콘을 실제 성 이름 라벨의 오른쪽 끝에 정렬했습니다.
- 성 문장과 그 아래의 병사수 표시는 기존 위치를 유지합니다.
