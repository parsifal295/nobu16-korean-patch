# NOBU16 PK 한글 패치 v0.14.0 릴리즈 노트

## 순정 Steam JP 직행 통합 패처

> **설치 전 필수:** v0.14.0은 **Steam JP 1.1.7 순정 파일**만 지원합니다.
> 이전 패치·모드·수동 수정 이력이 있거나 상태가 불확실하면, 반드시 Steam에서
> **NOBU16 → 속성 → 설치된 파일 → 게임 파일 무결성 확인**을 실행해 원본 상태로
> 복구한 뒤 설치하십시오.

- v0.14.0은 **순정 Steam JP 1.1.7에서 바로 적용**합니다. v0.13.x 선행 설치가
  필요하지 않습니다.
- 사용자가 실행할 파일은 `APPLY_KOREAN_PATCH.bat` 하나입니다. 정적 EXE와 한글
  리소스 15개를 한 흐름으로 적용합니다.
- 적용 전 원본 EXE와 15개 리소스의 해시를 검사합니다. 텍스트 10개는 고정 레저로
  재구성하고, 폰트·이미지 5개는 원본 검증형 BSDIFF40 델타로 복원합니다.
- 정적 EXE → 리소스 순서로 적용하며, 리소스 단계가 실패하면 이번 실행에서 순정
  EXE를 바꾼 경우 EXE도 자동으로 복원합니다.
- 전체를 순정 상태로 되돌릴 때도 `RESTORE_KOREAN_PATCH.bat` 하나만 실행합니다.
  리소스 → 정적 EXE 역순으로 복원하고, 최종 해시를 다시 검사합니다.
- ZIP에는 완전한 게임 리소스와 `NOBU16PK.exe`가 포함되지 않습니다.

### 설치 방법

1. Steam에서 NOBU16의 게임 언어를 **Japanese**로 설정합니다.
2. Steam 라이브러리에서 **NOBU16 → 속성 → 설치된 파일 → 게임 파일 무결성 확인**을
   완료해 순정 JP 1.1.7 상태를 준비합니다.
3. Steam과 게임을 종료한 뒤 ZIP의 **모든 파일**을
   `...\SteamLibrary\steamapps\common\NOBU16` 최상위에 풉니다.
4. `APPLY_KOREAN_PATCH.bat`를 실행합니다.
5. `패치 완료! Steam에서 게임을 시작하세요.`가 표시되면 Steam에서 게임을 시작합니다.

패처가 중단되면 파일을 수동으로 덮어쓰지 말고 Steam 무결성 확인을 다시 실행한 뒤
처음부터 설치하십시오.

## 번역·표기 보정

- Base·PK 인물 대사 B17 블록 17개 레코드의 24개 리터럴을 문장 연결, 조사,
  이름 경계 공백과 문법 구조 기준으로 보정했습니다.
- PK 이벤트 대사 20건(아네가와 전투 14건 포함)의 수동 줄바꿈을 문맥 단위로 다시
  배치했습니다. 문장 축약 없이 색상 태그·런타임 토큰·종료 코드를 보존했고,
  현재 30px/유효 폭 912px/최대 4줄 기준을 적용했습니다.
- 모든 무장 후리가나 칸을 한국 독음으로 전환했습니다. 독음은 `직전신장`처럼
  공백 없이 표시하고, 일반 화면의 무장명 표기는 변경하지 않았습니다.
- 동적 조합 성명이 붙어 보이던 전용 별칭 성 블록 5건은 성·이름 사이 공백을
  보정했습니다.

## 정적 변경

- 이벤트 메시지 글자 크기를 36px에서 30px로, 줄간격을 10에서 8로 조정했습니다.
- 이벤트 내레이션 부모 폭을 882px에서 972px로 넓혀 실효 텍스트 폭 912px을
  확보했습니다.
- 자동 줄바꿈의 주·대체 경로 한도를 전각 20자급에서 전각 30자급으로 조정했습니다.
- 지도 성 이름 뒤 상태 아이콘 및 보조 표시의 동적 정렬을 유지합니다.
