가독성을 위해 제어 태그는 다음처럼 표기했다. 실제 검사는 원래 `ESC C?` 토큰 순서를 비교했다.

- `‹A›` = `ESC CA`, `‹B›` = `ESC CB`, `‹C›` = `ESC CC`, `</>` = `ESC CZ`
- `⏎` = 수동 LF
