# 성 이름 ID 9151~9249 사람 검수 제안 v0.2

## 결론

ID `9151..9249` 99개를 공식 게임 EN 표시명, JP 표시명, JP 읽기 블록과 같은 ID로
대조했다. 89개는 현 초안을 유지하고, 10개는 별도 제안으로 남긴다. 원본
`public/castle_names_ko_9151_9542.v0.1.json`은 수정하지 않았다.

- ID `9210`: `갓산 도다` → `갓산토다`. EN 단어 경계를 기계적으로 보존한 결과를
  정착된 한국어 복합 지명으로 바로잡는 독립적인 고신뢰도 제안이다.
- 나머지 9개: 지명에서 일본어 `つ`를 `츠`가 아니라 표준 표기 `쓰`로 옮기는 제안이다.
  개인명 작업 스트림의 `츠` 정책과 충돌할 수 있으므로 `approve_geographic_tsu_style`
  결정 전에는 병합하지 않는다.

JSON 제안서는
`reviews/castle_names_ko_9151_9249.review-proposal.v0.2.json`이다.
