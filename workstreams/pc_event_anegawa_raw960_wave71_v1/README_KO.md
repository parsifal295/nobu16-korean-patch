# W71 아네가와 이벤트 수동 개행 재검수

대상은 PK MSG_PK/JP/msgev.bin의 아네가와 장면 56개(5777–5802, 5885–5914)다.

- 일본어 원문 개행은 사용하지 않는다.
- W70 한국어 문장은 축약·삭제·개명·내용 수정 없이 보존한다.
- 수동 개행은 한국어 문장과 어절의 의미 단위에서만 새로 지정한다.
- 실게임 화면으로 확인한 줄바꿈 기준은 원본 G1N 전각 48px·반각 24px, 한 줄 raw 960px 이하이다.
- static patch 007의 30px·줄간격 8은 표시만 바꾸며 이 위젯의 wrap 측정값을 30/48로 환산하지 않는다.
- 최대 4줄이며, 태그 내부 개행과 런타임 토큰 변경은 금지한다.

audit.v1.json은 각 줄의 표시 문자열, raw G1N 폭, 30/48 표시 환산 폭, 전각/반각 수, 줄 수, live raw 960px 초과 여부를 남긴다.
