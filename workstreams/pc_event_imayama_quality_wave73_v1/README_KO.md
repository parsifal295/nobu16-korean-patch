# W73 이마야마 이벤트 전수 검수

대상은 PK 이벤트 `5915–5937`의 23행이다. 이 작업은 Steam에 적용된 W71을
직접 수정하지 않는다. W72 오케하자마 후보의 `msgev.bin`을 고정 선행 입력으로
읽고, 그 위에 이마야마에서 확인한 5행만 덧쓴 private 후보를 만든다.

수정 행은 `5920, 5921, 5923, 5936, 5937`이며 나머지 18행은 JP·EN·SC·TC와
대조한 뒤 W72 문자열을 그대로 유지한다.

원문 일본어의 개행은 이식하지 않는다. 한국어는 원문 의미·정보·어조를 축약하거나
삭제하지 않고, 4줄 이내에서 문맥 단위로만 수동 개행한다. 색상 태그·제어 코드의
순서와 개수는 JP 원문 및 W72 입력과 일치해야 하며, 태그 안에는 개행을 넣지 않는다.

레이아웃 검사는 실제 PK 이벤트 경로의 raw G1N 기준을 사용한다.

- 전각: 48px
- 반각: 24px
- 한 줄 상한: raw 960px
- 최대: 4줄
- 보고용 실효 폭: `ceil(raw_g1n_width_px * 30 / 48)`

이 범위에는 런타임 인명 토큰이 없어 이름 예약 폭을 추정하지 않는다. 생성물은
`tmp/pc_event_imayama_quality_wave73_v1/` 아래에만 쓰며 Steam, Git 원격,
릴리스에는 접근하지 않는다.
