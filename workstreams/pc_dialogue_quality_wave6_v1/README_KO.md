# PC 대사 품질 Wave 6

이 작업은 현재 Steam의 Wave 5 전체 프로필을 입력으로 고정한 3개 정적 문장 정리입니다.

- `MSG_PK/JP/msggame.bin` `2:314`: 원문에 없는 `공격자:` 라벨 제거
- `MSG_PK/JP/msggame.bin` `2:315`: 원문에 없는 `담당자:` 라벨 제거
- `MSG_PK/JP/msggame.bin` `13:583`: 튜토리얼 해금 목록의 `명령:“증설”`을 `명령의 “증설”`으로 정리

근거는 PC 일본어 원문과 PC EN/SC/TC의 의미 대조뿐이며, Switch 한국어 텍스트는 사용하지 않습니다. 세 레코드는 현재 레코드 해시, 원문 레코드 해시, 불투명 제어 바이트, 수동 개행, 출력 레코드 해시와 전체 출력 프로필을 모두 고정합니다.

`13:583`은 고정 개행과 최대 폭이 정적으로 유지되지만 튜토리얼 UI의 실제 컨테이너 표시는 별도 실게임 QA 대상으로 남습니다. 이 작업은 그 시각 검증이 완료됐다는 주장을 하지 않습니다.
