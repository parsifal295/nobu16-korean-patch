# Steam JP PK `/18` 비로고 UI 라벨 렌더 준비

대상은 `RES_JP_PK/res_lang_pk.bin /18`의 투명 43개 UI/정보/시나리오 편집 라벨뿐이다. `/3`, `/24`, 로고·타이틀·브랜드 아트와 Switch 픽셀은 영구 제외한다.

이 스트림은 PC JP의 활성 한글 글리프 경로(`/16`, table 0, 48px)를 이용할 렌더 계획과 source-free 한국어 문구·증거 템플릿만 준비한다. 아직 슬롯별 실제 화면 trace와 BC3 block-aligned 텍스트 사각형 QA가 없으므로 후보 리소스를 생성하지 않으며 `build` 명령도 의도적으로 실패한다.

- 확정 문구: 40개
- 용어 검토 필요: 슬롯 1, 2, 19
- 화면 trace/텍스트 사각형 QA 필요: 43개 전부
- 게임 설치·릴리즈·GitHub 쓰기: 금지
