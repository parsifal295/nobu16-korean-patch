# msggame.bin 번역 배치 19 v0.19

`MSG_PK/SC/msggame.bin` 좌표 기반 한국어 초벌 번역의 후속 배치다.
공개 산출물에는 원문이나 완성 게임 리소스를 넣지 않고, 좌표·SC UTF-16LE
SHA-256·한국어 번역문만 기록한다.

| 다음 연속 좌표 | `(6, 3478, 0)` |
| 런타임 검수 | 미실시 |

- 조정 주청·칙명 강화, 군단 해임·추방, 혼인 동맹 파기, 후계 계승,
  지행 무장 자동 임명, 공훈 일위 대사를 포함한다.
- SC·JP·EN·TC의 동일한 블록·레코드 전체 문맥을 비교했다. 언어별 리터럴
  분할 수가 달라질 수 있으므로 다른 언어의 `literal_id`를 직접 대응시키지
  않는다.
- 레코드 `3466`·`3467`·`3471`은 SC의 끝 슬롯이 빈 문자열이다. 각 레코드의
  표시 리터럴 4개만 번역하고 빈 슬롯은 `None`으로 보존한다.
- 동적 인물·세력·관직·수치 결합과 줄바꿈은 실제 화면에서 재검수해야 한다.

## 새 산출물

1. `build_translation_batch19.py`: 고정된 SC·JP·EN·TC 설치본을 읽기 전용으로
   검증하고 공개 오버레이와 오프라인 재패킹 검증 결과를 결정적으로 생성한다.
2. `public/msggame_ko_system_messages_b06r3302_3477.v0.19.json`: 원문 없는
   좌표 기반 번역 오버레이.
3. `evidence/translation_alignment_evidence.v0.19.json`: 원문 없이 기록한
   언어별 같은 레코드 정렬 근거.
4. `review/translation_review_index.v0.19.json`: 좌표별 사람·화면 검수 상태.
5. `translation_validation.v0.19.json`: 재현성·바이너리 구조·안전성 검증 결과.
6. `tests/test_translation_batch19.py`: 범위·빈 슬롯·원문 부재·오버레이 충돌·
   A/B 결정성·재패킹 회귀 테스트.

## 재생성 및 좁은 검증

```powershell
python -B workstreams/msggame/build_translation_batch19.py `
  --out-root workstreams/msggame
python -B -m unittest workstreams.msggame.tests.test_translation_batch19 -v
```

생성기는 설치된 SC·JP·EN·TC 파일의 packed/raw 크기와 SHA-256을 먼저
검사한다. 설치 파일, 폰트, 설치기, 실행 파일, 메모리, 레지스트리는
변경하지 않는다.
