# v0.6.0 — Steam PK v1.1.7 일본어 경로판

Steam PK `Update 1.1.7`(Steam build `18823764`) 순정 일본어 리소스 구조에서
직접 재빌드한 첫 정식 Steam 배포본이다. 공식 런처의 언어를 `日本語`로 선택해
사용한다.

## 설치 파일

- 자산: `NOBU16_PK_Korean_Patch_Steam_1.1.7_v0.6.0.zip`
- 크기: `238,184,085` bytes
- SHA-256: `1762265CD011D8476B8B90C87E3CA3F5F1D920B46C4A97460147A5FEEDA373D0`
- 내용: 원래 폴더와 파일명을 유지한 게임 적용 파일 10개

ZIP의 `MSG`, `MSG_PK`, `RES_JP`, `RES_JP_PK` 폴더를 게임 설치 폴더에 그대로
덮어쓴다. 두 `RES_JP*` 글꼴 컨테이너가 포함되어 있으므로 별도 폰트 설치는
필요하지 않다.

- 적용 후 목표 파일 해시 10개 일치

- 순정 predecessor 백업 기준 구조·폰트·ZIP 테스트 39 / 39 통과

- 타이틀의 `마우스를 클릭하십시오.`와 한글 메인 메뉴 실기 확인
- SC 리소스, 메모리 패치, DLL 주입, 후킹, EXE·레지스트리 변조 미사용
