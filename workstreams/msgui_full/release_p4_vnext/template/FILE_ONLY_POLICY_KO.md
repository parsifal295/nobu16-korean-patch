# 파일 전용 배포 정책

이 패키지는 게임이 종료된 상태에서 디스크의 `MSG_PK/SC/msgui.bin`과 `RES_SC/res_lang.bin` 두 파일만 레시피로 재구성합니다.

- 메모리 패치, 프로세스 부착, DLL 주입, 후킹, 프록시 DLL을 사용하지 않습니다.
- 게임 실행 파일과 공식 런처를 읽거나 수정하지 않습니다.
- 레지스트리를 읽거나 쓰지 않으며 게임을 자동 실행하지 않습니다.
- 전체 원본/수정 상용 리소스는 포함하지 않습니다. 번역 레시피와 프로젝트가 OFL 글꼴에서 만든 글리프 픽셀만 포함합니다.
- 전체 메시지 레시피는 5,100개 카탈로그 항목 가운데 실제 변경 작업 3,836개만 담습니다.
- Font-v4는 원시 문자 수요 645자에서 UI 제어·명령·아이콘 토큰 20자를 제외한 실제 수요 625자(한글 524자)를 검증합니다. 원본으로 충족되는 비한글 62자는 보존하고, 래스터/테이블1에는 563자, 테이블0에는 한글 524자와 전각 물음표를 합친 525자를 추가합니다.
- 공개 파일 경로는 고정 allowlist와 manifest 실제 트리의 완전 일치로 검사하며, 완성 리소스의 알려진 크기·해시·파일 시그니처, 추가 JSON 필드, 중첩 압축 파일을 거부합니다.
- JSON 객체 키는 raw UTF-8 단계에서 재귀적으로 검사합니다. escape 해제 후 같은 키와 `OrdinalIgnoreCase` 대소문자 충돌도 `ConvertFrom-Json` 전에 거부합니다.
- 패키지 내부 검증기의 신뢰는 외부 배포 페이지에 게시된 ZIP SHA-256과 별도 `.zip.sha256` 대조에서 시작합니다.
- 지원 원본 및 결과 SHA-256이 정확히 일치할 때만 작업합니다.
- 원본 두 파일은 게임 폴더 아래 `KR_PATCH_BACKUP/msgui_full_font_v4_v0_3`에 검증된 상태로 보관합니다.
- 같은 볼륨의 임시 파일과 거래 기록을 사용하며, 중단 후에는 다음 실행에서 원본 한 쌍으로 안전하게 복구합니다.

언어 선택은 사용자가 공식 런처에서 직접 **Simplified Chinese(간체 중국어)** 로 설정해야 합니다.

런타임 검증 대상은 Steam에서 받은 설치본이 아닌 비 Steam PC 설치본입니다. Steam 설치 경로나 Steam 실행 옵션을 전제로 하지 않습니다.

성 이름 세로 표시는 알려진 제한사항이며 출시 차단 조건이 아닙니다. 실제 SC 런타임 QA와 원본 복구 증거가 완료되지 않은 빌드는 `release_eligible=false`를 유지합니다.
