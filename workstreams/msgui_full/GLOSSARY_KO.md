# 신생 PK 한글화 핵심 용어집 (초안)

이 문서는 영문판 표기가 원뜻과 다를 때 일본어/중국어 원문을 우선해 번역하기 위한 작업 기준이다. 번역 검수 전까지 버전은 초안으로 유지한다.

| 일본어/한자 | 영문판 예 | 한국어 기준 | 비고 |
|---|---|---|---|
| 勢力 | Clan | 세력 | 가문과 혼용하지 않음 |
| 武将 | Officer | 무장 | 장수보다 게임 UI 일관성 우선 |
| 城 | Castle | 성 | 성채로 늘리지 않음 |
| 軍団 | Province | 군단 | 영문판 Province는 오역성 표기 |
| 郡 | County | 군 | 군단과 구분 |
| 国衆 | Tribe | 국인중 | 고유 제도명 |
| 家宝 | Treasure | 가보 | 보물로 번역하지 않음 |
| 具申 | Submission | 건의 | 문맥에 따라 건의/건의안 |
| 進言 | Counsel | 진언 | 具申과 구분 |
| 威風 | Authority | 위풍 | 시스템 고유명 |
| 知行 | Dominion | 지행 | 시스템 고유명 |

| 領主 | Land Holder | 영주 | 성주와 구분 |
| 城主 | Lord | 성주 | 영주와 구분 |

## UI 문체

- 버튼은 가능한 한 명사형 또는 짧은 동작형으로 통일한다: `전체 선택`, `닫기`, `폴더 열기`.
- `%d`, `%s`, 폭 지정자, ESC 색상 코드, PUA 아이콘은 순서까지 보존한다.
- `translated`는 초벌 번역, `reviewed`는 화면과 문맥까지 확인한 번역을 뜻한다.
- 영문판의 `Province`처럼 일본어 원문과 불일치하는 항목은 일본어/중국어 공통 의미를 따른다.
