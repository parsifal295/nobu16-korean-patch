- PK `10908`은 원문의 토벌 결과를 정확히 반영하도록 `패잔병을 잡았다`를
  `패잔병을 베었다`로 고친다. 수동 줄바꿈과 3줄 구조는 보존한다.
- PK `14522`의 일본어 음역 제목을 같은 PC 원문군에 이미 정착한 표기
  `시대 개요(덴쇼 10년)`으로 복원한다.

  Base는 렌더러 폭 비율이 미해석이지만 이 후보는 기존과 동일한 줄바꿈·폭이다.

  생성한다. Steam 적용·Git·네트워크·릴리즈 기능은 없다.

이번 후보에는 줄바꿈을 옮기는 이벤트 행을 넣지 않는다. 해당 행은 동일한 활성 글꼴
