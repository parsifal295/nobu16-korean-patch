# `msgui` 좌표·번역 품질 수정

현재 Steam PC `MSG_PK/JP/msgui.bin`의 한글을 순정 PC 일본어 원문과 1:1로 대조해, 일본어 잔류와 잘못된 좌표에 들어간 한국어 문장을 함께 고친다.

- 스위치판 한글은 읽거나 품질 기준으로 사용하지 않는다.
- 교정문은 반드시 같은 좌표의 순정 PC 원문과 printf·ESC·런타임 토큰·개행·공백·아이콘 프로필이 일치해야 한다.
- 공개 오버레이에는 한글·해시만 저장한다. 원문과 완성 게임 파일은 `tmp` 밖으로 내보내지 않는다.
- 후보 생성은 Steam 설치본을 읽기만 하며 쓰지 않는다.

검수안이 현재 한글을 `ko`, 교정문을 `proposed_ko`에 함께 보관하는 경우에는 현재 좌표 문자열 또는 현재 해시를 반드시 다시 대조한다. 같은 방식으로 순정 PC 일본어 해시가 제공되면 해당 좌표와 재대조하므로, 짧은 라벨의 좌표 밀림도 단순 문자열 치환으로 통과하지 않는다.
