# PC B15 고위험 의미 재점검 v1

## 범위와 독립성

- 대상은 PC `msggame.bin`의 블록 15(B15)뿐이다.
- 아래의 Base/PK 한국어 파일과 정확한 pristine PC 일본어 파일 4개만 직접 대조했다.
- 기존 B15 보고서·후보, Switch/EN/SC/TC, 다른 workstream의 결과는 읽지 않았다.
- 명백한 의미 반전, 용어 오역, 문법 오류만 기록했다. 런타임 토큰·폭·표시 맥락이 필요한 것은 Hold로 남겼다.
- 후보 파일 생성, Steam 적용, Git 변경, 네트워크/푸시/커밋/릴리즈는 수행하지 않았다.

`⏎`는 해당 literal 안에 이미 있는 수동 LF 1개를 뜻한다.

## 고신뢰 수정 방향 — 4개 이슈군, 7 literal

| 이슈 / 좌표 | JP 원문 | 현재 KO | 판정 및 수정 방향 |
| --- | --- | --- | --- |
| **馬術을 마술로 오역**<br>Base `B15:1875:1`, `B15:1890:1` | `⏎馬術の訓練を行い⏎兵を強化し` | `⏎마술 훈련을 실시해⏎병사를 강화하고` | `馬術`은 승마·마술(魔術)이 아니다. `⏎승마 훈련을 실시해⏎병사를 강화하고`로 수정. 정확히 같은 JP 원문의 PK `B15:1905:1`, `B15:1920:1`도 이미 이 문구다. |

수정 방향의 literal 안전성도 별도로 확인했다. 7개 좌표 모두 현재 literal과 목표 literal의 수동 LF 개수가 같고, `C0` 제어 바이트 수 역시 변하지 않는다.

| 좌표군 | 현재 → 목표 LF | 현재 → 목표 C0 |
| --- | --- | --- |
| Base `1875:1`, `1890:1` | 2 → 2 | 0 → 0 |
| Base `2348:0`, PK `2379:0` | 1 → 1 | 0 → 0 |
| PK `2257:1`, `2258:1` | 0 → 0 | 0 → 0 |
| PK `2279:0` | 1 → 1 | 0 → 0 |

## Hold — 후보/적용 전에 실제 게임으로 확인할 것

- PK `B15:2257:1`, `B15:2258:1`은 도시명 등 런타임 값 뒤에 붙는 조각이다. 의미 수정 방향은 Base의 정확한 동등 원문으로 확인됐지만, 실제 도시명과 결합한 문장·폭은 게임에서 확인한 뒤에만 적용한다.
- PK `B15:2279:0`의 JP literal은 `でき`에서 끝난다. 따라서 이 literal 단독이 아니라, 정확히 같은 JP literal과 같은 대화 구조를 가진 Base `B15:2249:0`의 KO 부정형을 근거로 판정했다. 적용 시 해당 외교 제한 상태를 실제 게임에서 확인한다.
- B15의 동적 조사/목적격 조각과 UI bullet `・`는 문맥·레이아웃 성격이라 일괄 치환하지 않았다.

## 결론

이번 결과는 B15의 고위험 오류만 좁혀 기록한 재점검이며, B15 전체 또는 인물 대사의 전수 품질 완료를 뜻하지 않는다. 이 workstream에는 후보 파일이 없고 Steam·Git·릴리즈 상태도 변경하지 않았다.
