# 인물 대사 런타임 종결어미 핫픽스

`msggame` 일부 레코드는 일본어 활용 종결을 리터럴 밖의 바이트코드로 붙입니다. 기존 한글화는 그 앞의 일본어 어간만 한글 어간으로 바꾸고 종결 바이트코드를 유지해, 게임 안에서 `크게 나오했습니다군`처럼 비문이 만들어졌습니다.

이 작업은 해당 조합을 유지하지 않습니다. 문제 레코드를 완결된 한국어 대사 하나로 다시 구성해 일본어 종결 바이트코드를 제거합니다.

- 기본 게임: `6:2288`, `6:2289`, `13:17`, `13:142`
- PK: `6:2295`, `13:17`, `13:142`

PK `6:2295`는 앞선 질문 대사를 그대로 복제한 별도 품질 오류였다. 원문 PC 일본어의 독립된 응답 대사에 맞춰 별도 응답으로 복원한다.

`build_pc_dialogue_runtime_suffix_hotfix_v1.py`는 처음 적용 전 상태와 런타임 종결어미 수정 후 상태를 각각 두 메시지 파일 해시로 고정 검증하고, Steam 설치본을 직접 수정하지 않는 11파일 후보 트리를 만듭니다. 후보 트리에서 바뀔 수 있는 파일은 다음 두 개뿐이며, 후속 응답 교정 단계에서는 PK 파일 하나만 바뀝니다.

- `MSG/JP/msggame.bin`
- `MSG_PK/JP/msggame.bin`

나머지 아홉 개 파일은 바이트 단위로 유지되어, 파일 전용 트랜잭션에서 `retain`으로 기록됩니다.
