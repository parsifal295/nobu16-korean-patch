# W72 오케하자마 이벤트 원문 재검수

대상은 Steam PC PK `MSG_PK/JP/msgev.bin`의 오케하자마 장면
`4494–4510`이다. 현재 Steam Korean, 순정 PC 일본어, PC EN/SC/TC를 다시
대조해 17행 전부를 읽었고, 실제 바이트 변경은 11행이다.

```text
4494, 4495, 4496, 4498, 4502, 4503, 4504, 4506, 4507, 4508, 4509
```

## 교정 원칙

- 일본어 원문 LF는 그대로 이식하지 않는다. 한국어 문맥과 어절 단위로 새로
  수동 개행한다.
- 원문의 사건·수식어·인물·관계를 축약하거나 삭제하지 않는다.
- ESC 색상 태그, 런타임 토큰, printf와 제어 코드의 순서·개수는 보존한다.
- 이벤트 위젯은 raw G1N 전각 48px·반각 24px, 한 줄 960px, 최대 4줄로
  검사한다. 30/48 표시 환산값은 보고용이다.
- `[b1448]`은 이 장면에서 실제 표시되는 `나가오 가게토라`(360px)로만
  예약한다. 다른 장면에 전역 적용하지 않는다.

핵심 수정은 다음과 같다.

- `갑상준 삼국 동맹` → `가이·사가미·스루가 삼국 동맹`
- `또한,`처럼 원문에 없는 접속을 삭제하고 영향 관계를 복원
- `海道一の弓取り`·`今川治部大輔` 및 4507 비문의 의미를 원문대로 재번역
- 4508에서 누락된 `일개 다이묘 오다 노부나가`와 `…뿐 아니라`를 복원

`audit.v1.json`은 17행 모두에 대해 표시 문자열, raw G1N 폭, 30/48 환산 폭,
전각/반각 문자 수, 줄 수, 960px 초과 여부와 원문·기존·대상 해시를 남긴다.

이 workstream은 private 후보만 만든다. Steam 적용·푸시·릴리스 기능은 없다.
