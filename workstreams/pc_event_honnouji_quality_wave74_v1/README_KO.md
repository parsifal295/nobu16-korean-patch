# W74 혼노지 이벤트 대사 품질·개행 검수

대상은 PK 이벤트 `10990–11009`의 20행이다. 이 작업본은 커밋된 W73
(`pc_event_imayama_quality_wave73_v1`)의 `tmp/.../candidate-final/MSG_PK/JP/msgev.bin`을
엄격한 선행 입력으로 읽는다. 선행 파일의 압축본·원본 해시와 크기가 고정 프로필과
일치하지 않으면 빌드하지 않는다.

수정 대상은 `10990, 10995, 10996, 10998, 10999, 11000, 11003, 11004, 11006,
11008, 11009`뿐이다. 나머지 9행은 W73의 문자열을 바꾸지 않는다. 직접 PC 일본어는
의미와 제어 코드 대조에만 사용하며, 일본어 원문의 개행 위치는 이식하지 않는다.

각 변경행은 태그·제어 코드 순서를 보존하고, 한국어 문장을 축약하거나 정보를 삭제하지
않은 상태에서 문맥 단위로 수동 개행한다. 태그 내부에는 개행을 넣지 않는다.

보고서는 각 표시 줄의 원본 G1N 폭(전각 48px, 반각 24px), `ceil(raw * 30 / 48)`
실효 폭, 전각/반각 수, 960px 엄격 검수 초과 여부와 static-patch-007의 912px 실효 폭
초과 여부를 남긴다. 모든 줄은 4줄 이하이며 raw 960px 이하로 검수한다.
