# Wave 48 정적 UI 0143 제거 후보

이 작업은 현재 Steam PC W45 대화 리소스의 정적 UI 문장 16개 계열, Base/PK 합계 32개 물리 레코드만 비공개 후보로 재조립한다. 정확한 W45 입력과 PC 원문(JP/EN/SC/TC)을 다시 대조하여 기존 target pin을 재확인했다. Nintendo Switch 한국어 파일은 읽지 않는다.

각 대상은 일본어 문장 끝에 남은 정적 01 43 굴절 명령 하나만, 지정된 바이트 오프셋에서 제거한다. 02xx 런타임 토큰과 01 43 01 00 00 00 런타임 슬롯은 대상에서 모두 부재해야 하며, 하나라도 나타나면 빌드가 중단된다. literal marker topology와 05 05 05 terminator는 보존한다.

입력은 다음 두 Steam PC 패킹 프로필과 정확히 같아야 한다.

- Base MSG/JP/msggame.bin: F9342D73DE50FDFC97C1F8365A20FD5CEABD024CE63B82AF1F112D5EDEDCFCBB, 1,504,410 bytes
- PK MSG_PK/JP/msggame.bin: 0A92516BC4B0A7AE98FD66418AD0BE289682B9DEE2CB25A8A1740A9609288092, 1,806,538 bytes

32개 전부 수동 줄바꿈 하나를 가진 정확히 2줄이며, 활성 PC JP 글꼴 기준 최대 폭은 840px로 912px 제한 이하이다. 빌더는 각 레코드의 현재/JP/목표 SHA-256, 현재/목표 size, 0143 제거 오프셋, opaque spans와 각 줄 폭을 pin으로 검사한다.
