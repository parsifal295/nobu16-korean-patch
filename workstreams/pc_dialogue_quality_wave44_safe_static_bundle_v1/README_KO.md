# Wave 44 보수적 인물 대사 통합 private 후보

현재 Steam PC 기준선에서 정적 대사 51건(Base 13, PK 38)을 레코드 단위로 합성한다. 각 포함 항목은 수동 개행 수, literal 경계, opaque 바이트, 마커 구조, 종료 코드를 보존하며 912px 경계에 도달하는 항목은 넣지 않는다.

실게임 화면 QA가 필요한 7건과 Base UI 장문 2건, 단순 명료성 개선 1건은 의도적으로 제외한다. 이 작업물은 `tmp/` 아래 private 후보만 만들며 Steam 적용, Git 조작, 네트워크, 릴리즈 기능은 없다.
