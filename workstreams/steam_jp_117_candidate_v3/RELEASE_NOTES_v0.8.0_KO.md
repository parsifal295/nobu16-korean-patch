# v0.8.0 배포 메모

대상은 Steam PK `1.1.7`(Steam build `18823764`) 일본어(`JP`) 경로다.
메모리 패치, DLL 주입, 후킹, EXE·레지스트리 변조와 SC 리소스를 사용하지 않는다.

## 번역 갱신

- 공통 메시지 5종의 실제 한국어 변경: 40,581개
- wave08 검토 의미 항목 94개 전부 처리
- Steam 일본어판 장수 성씨 980개 복구
- 대표 장수명 검증: `오다 ` + `노부나가` = `오다 노부나가`
- 실제 변경과 분리한 원본 동일 구조 계약: 1,796개
- 추가 검토 대기: 792개(형식 계약 차단 730개 + 정렬 간극 62개)
- 원문 유지 내부 더미 2개는 정렬 간극 62개에 포함
- 공용 메시지 전체 좌표 검산: 40,581 + 1,796 + 730 + 62 = 43,169
- `MSG_PK/JP/msggame.bin`: v0.7.0과 동일한 28,272개 적용
- `MSG_PK/JP/msgui.bin`: v0.7.0과 동일한 4,036개 매핑, 3,955개 실제 변경
- `MSG/JP/strdata.bin`: v0.7.0과 동일한 24,524개 적용

## 폰트와 산출물 구조

- 서울한강 EB 48px·96px급, 서울한강 B 32px·64px급 유지
- `RES_JP`, `RES_JP_PK`, `RES_JP_PK_PORT`의 검증된 4개 폰트 후보를
  v0.7.0과 바이트 단위로 동일하게 유지
- 원래 Steam 폴더 구조와 파일명을 보존한 JP 경로 exact-12 ZIP
- 고정 파일명: `NOBU16_PK_Korean_Patch_Steam_1.1.7_v0.8.0.zip`
- `RES_JP_PK_PORT/res_lang_pk_port3.bin`과 모든 SC 경로 제외

## 현재 상태

소스 없는 재현 빌더와 검증 핀을 준비했고, 임시 exact-12 재현과 핀 비교를
통과했다. 재현 ZIP은 356,142,573바이트이며 SHA-256은
`5F2E6641BDC7BCF615CA7AC697881DA94E61CA95CAE147A053B2C0DFD871F8E9`다.
검증 파일 SHA-256은
`7B05C0F44AE4849BAB65E55FD32CF62B37931E66EE41D6CC743BDC083F7D1FC8`다.

임시 후보와 ZIP은 검증 직후 삭제했다. 아직 Steam 설치 적용, 게임 화면 검증,
보존 ZIP 빌드 또는 GitHub 릴리스는 수행하지 않았다.
