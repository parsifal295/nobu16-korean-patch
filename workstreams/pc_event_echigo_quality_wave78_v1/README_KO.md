# W78 에치고 출가 선언 이벤트 품질·개행 검수

이 작업 스트림은 PK 이벤트 ID `4280`~`4314`의 35행을 전수 검수한다. 입력은 반드시 온디스크 W77 후보
`tmp/pc_event_komaki_nagakute_quality_wave77_v1/candidate-final/MSG_PK/JP/msgev.bin`이며, 파일 범위와
압축·원시 프로필이 고정값과 일치하지 않으면 후보를 만들지 않는다.

변경 13행은 다음과 같다.

`4281, 4288, 4289, 4290, 4294, 4296, 4298, 4299, 4302, 4303, 4304, 4313, 4314`

나머지 22행은 직접 PC JP 및 PC EN/SC/TC 문맥 대조 뒤 유지한다. 일본어 원문의 개행은 이식하지 않으며,
한국어 개행은 의미 단위로만 배치한다. 문장 축약·정보 삭제·인명 변경은 하지 않는다.

각 줄은 원본 G1N 전각 48px·반각 24px으로 측정해 raw 960px 이하, 최대 4줄로 제한한다. 감사 보고서에는
표시 문자열, raw 폭, `ceil(raw * 30 / 48)` 실효 폭, 전각/반각 문자 수 및 초과 여부를 기록한다. 색상 태그·런타임
토큰·종료 구조는 보존하며 태그 내부에는 개행을 넣지 않는다.

`[b1448]`, `[bm1448]`, `[bs1448]`는 이 장면에 한정한 보수적 폭 예약이다. 실제 런타임 표시를 증명한 값이
아니므로 모든 예약과 감사 결과는 `runtime_proven: false`로 기록한다.

후보 산출물은 `tmp/pc_event_echigo_quality_wave78_v1/candidate-final/` 아래에만 생성한다. Steam 적용, Git
커밋·푸시, 릴리스는 이 작업 범위에 포함하지 않는다.
