# PK msggame 일본어 경로 전환 검증

이 작업 묶음은 누적 한국어 `msggame` 오버레이를 일본어 경로용 후보로 만드는 두 방식을 검증한다.

- native 후보: 일본어 컨테이너를 유지하며 직접 호환 항목과 유일성이 증명된 항목만 적용한다.
- mirror 후보: 간체중문 컨테이너에 누적 오버레이 전체를 적용한 뒤 일본어 리소스 경로용 후보로 만든다.
- private 출력은 명시한 staging root 아래 `native/MSG_PK/JP/msggame.bin`과 `mirror/MSG_PK/JP/msggame.bin` 두 파일만 허용한다.

현재 검증 범위는 파일 구조, 원본 해시, 좌표, 제어 토큰, 재빌드, 비대상 불변성과 결정적 후보 해시까지다. 실제 설치 게임에는 적용하지 않았고 게임 실행, 화면 캡처, 메뉴 이동 검증도 하지 않았다. 따라서 두 후보 모두 런타임 미검증 상태이며 릴리스 준비 완료로 취급하면 안 된다.
