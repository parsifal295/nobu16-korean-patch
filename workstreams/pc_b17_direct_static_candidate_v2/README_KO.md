# PC B17 정적 번역 보정 후보 v2

현재 W45 Steam PC 한국어와 깨끗한 Steam PC 일본어만 직접 대조해 만든 사설 후보이다. Switch판이나 다른 언어판은 열거나 근거로 사용하지 않았다.

이 후보는 **44 literal / 44 record**만 바꾼다.

- `MSG/JP/msggame.bin`: 4 literal / 4 record
- `MSG_PK/JP/msggame.bin`: 40 literal / 40 record

모든 변경은 현재 W45 한국어 전문(preimage), 바꿀 한국어 전문, PC 일본어 전문을 정확히 고정했다. 전체 원문은 [빌더](build_pc_b17_direct_static_candidate_v2.py)의 `TARGETS` 및 생성되는 `audit.v2.json`에 남는다. 넓은 문자열 치환은 사용하지 않는다.

## 의도적으로 제외한 범위

- 수동 LF(줄바꿈)는 하나도 추가·삭제·이동하지 않았다.
- 동적 이름/조사, 런타임 토큰, `Color.Blue` 같은 색상 마크업, 진행 수치 괄호는 후보에서 제외했다.
- Base `17:5`는 동적 이름/조사 HOLD가 섞인 레코드라 전체를 제외했다.
- PK HOLD literal은 `17:226:*`, `17:282:0`, `17:510:*`, `17:920:*`, `17:991:*`와 빌더의 `EXCLUDED_HOLD_SLOTS`에 적힌 모든 진행 수치/괄호 슬롯이다.
- 따라서 이 후보는 이벤트 줄바꿈 재배치나 런타임 조립 방식 변경을 해결하지 않는다.

## 재현과 검증

후보 산출물은 저장소가 아니라 `tmp/pc_b17_direct_static_candidate_v2/candidate` 아래에만 생성된다. 빌더에는 Steam 적용, 트랜잭션, Git, 네트워크, 푸시, 릴리스 기능이 없다.

```powershell
py -3 -B -X utf8 workstreams\pc_b17_direct_static_candidate_v2\test_pc_b17_direct_static_candidate_v2.py
py -3 -B -X utf8 workstreams\pc_b17_direct_static_candidate_v2\build_pc_b17_direct_static_candidate_v2.py build
py -3 -B -X utf8 workstreams\pc_b17_direct_static_candidate_v2\build_pc_b17_direct_static_candidate_v2.py verify-private
py -3 -B -X utf8 workstreams\pc_b17_direct_static_candidate_v2\build_pc_b17_direct_static_candidate_v2.py diff-check
```

검증은 W45 입력·PC 일본어 근거·후보 출력의 packed/raw SHA-256을 모두 고정하고, B17 literal topology, 전체 archive의 target-only 차이, 변경 literal의 LF/제어 바이트 보존, opaque skeleton 보존을 확인한다.
