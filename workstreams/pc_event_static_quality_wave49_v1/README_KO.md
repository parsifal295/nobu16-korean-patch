# Wave 49 정적 이벤트 품질 후보

이 작업은 이미 적용된 W45 Steam PC MSG_PK/JP/msgev.bin을 입력으로 고정해, 구조와 폭이 정적으로 안전한 이벤트 33개만 private 후보로 만드는 빌더다.

- 입력 W45 SHA-256: 01287E2ECC5328C85348657EFF06553353CB8664B0FB7E1669DB9FC591D53EBE
- 출력 후보 SHA-256: AC9C0F7FE72ADA6FA4604C1359A3FFA155BB5C166A590C3FC77BAD7C390CC90B
- 출력 위치: tmp/pc_event_static_quality_wave49_v1/candidate/
- Steam 적용, 파일 트랜잭션, Git, 네트워크, 푸시, 릴리즈 기능은 구현하지 않았다.

## 포함 범위

다음 33개 PK 이벤트 슬롯만 변경한다.

3240, 3897, 3949, 3951, 3953, 3954, 3957, 6235, 6343, 6419, 6724, 6863, 6911, 6913, 6925, 6929, 7033, 7332, 7525, 7559, 7562, 7577, 7940, 7985, 8026, 8032, 8615, 8626, 9148, 9447, 9470, 9491, 9540

각 슬롯은 다음을 모두 검증한다.

- W45 입력 파일과 raw 테이블 프로필
- legacy PC JP 원문 좌표와 원문 anchor
- 현재/목표 UTF-16LE 해시를 포함한 33행 결합 해시
- 런타임 토큰, 색상 태그, 수동 LF 토폴로지 불변
- 최대 3줄 및 줄당 912px 이하
- 재압축 출력의 packed/raw SHA-256

## 의도적으로 제외한 hold

- 의미 재번역·재개행 hold 25개: 3956, 5273, 5564, 5654, 5790, 5794, 5870, 6408, 6558, 6584, 6783, 7236, 7312, 8162, 8726, 9340, 9347, 9423, 9874, 10595, 10961, 10963, 10966, 10967, 11003
- 색상 태그 내부 LF QA hold 46개: 전부 삭제하지 않는다. 39개는 외부 재개행 가능성만 진단됐고, 3202, 3900, 3934, 4140, 8723, 9359, 10045는 문장 축약이 필요하다.
- 색상 escape 역할 QA hold 25개: 9887, 9916, 9998, 10032, 10274, 10276, 10288, 10299, 10325, 10381, 10386, 10450, 10483, 10484, 10499, 10513, 10566, 10625, 10630, 10788, 10815, 10854, 10934, 10940, 10943
- 이벤트 밖 인명 hold: 291, 292, 293의 이노우에 계열 표기는 NPC/인명 정비 범위에서 따로 다룬다.

## 실행

    python .\build_pc_event_static_quality_wave49_v1.py build
    python .\build_pc_event_static_quality_wave49_v1.py verify-private
    python -m unittest -v .\test_pc_event_static_quality_wave49_v1.py

build는 기존 private 후보를 덮어쓰지 않는다. 후보를 다시 만들려면 먼저 검증하고, 별도 승인된 private 정리 절차를 거쳐야 한다.
