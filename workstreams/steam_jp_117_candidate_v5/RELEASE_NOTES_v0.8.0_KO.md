# v0.8.0 배포 메모

Steam PK 1.1.7 일본어 경로용 v0.8.0 배포 검증 메모입니다. 실제 Steam 설치 및 사용자 화면 한글 출력을 확인한 exact-14 파일 벡터만 정식 릴리스에 포함합니다.

## v5에서 보강한 실행 경로

기존 12파일 조합은 `MSG_PK/JP` 파일 위주였습니다. 실제 PK 실행 화면에서는 기본판 `MSG/JP`의 일부 대사도 함께 로드되므로 v5는 아래 두 파일을 추가했습니다.

- `MSG/JP/msggame.bin` — 지도 튜토리얼·조언 대사 22,924개 전송 계약
- `MSG/JP/ev_strdata.bin` — 이벤트 대사 13,045개 전송 계약

따라서 v5는 **exact-14 파일**만 교체합니다. 다른 일본어 파일, SC 파일, EXE, 레지스트리, 메모리는 변경하지 않습니다.

## 포함된 기존 화면 수정

- 시나리오 선택 기간/월 표기
- 이벤트 목록의 가문명 표기
- 전법 목록 독음 표기
- 관직 표시명과 일본어 독음 분리
- `MSG_PK/JP/msgev.bin` 잔여 이벤트 문장·분기
