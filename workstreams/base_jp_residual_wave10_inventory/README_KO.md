# 본편 JP 잔여 대사 Wave 10 인벤토리

이 작업은 Steam 1.1.7의 본편 JP `msggame`과 `ev_strdata`에서, Switch v1.3의 엄격 전송 뒤에 남은 377개 좌표를 분류한다. 현재 후보, 스팀 게임 파일, 배포 압축 파일은 변경하지 않는다.

## 결과

- 실제 화면에 보이는 행: 318개
- 바로 다음 오버레이에 넣을 수 있는 형식 검증 완료 행: 310개
- 용어 표기 검토가 필요한 한자 병기 행: 8개
- 글머리표 같은 비문자 형식 토큰: 59개

본편 `msggame` 332개 중 207개는 이미 한국어로 준비되어 있었지만 가운데점 문자를 넓게 일본 문자로 판정한 전송 조건 때문에 제외됐다. 62개는 앞뒤 공백만 원문 형식에 맞추면 되고, 1개는 퍼센트 기호 형식만 맞추면 된다. 나머지 3개는 한자 병기 용어 검토 대상이다.

본편 `ev_strdata` 45개는 모두 이벤트 서사나 장면 표시에 해당한다. 이 중 40개는 가운데점 판정만 해소하면 바로 전송할 수 있고, 5개는 한자 병기 용어 검토 대상이다.

## 다음 적용 순서

1. `public/base_jp_residual_wave10_contracts.v1.json`의 310개 계약을 새 후보 조립 단계에서만 적용한다.
2. 지도 도움말, 전략 도움말, 조언 및 이벤트 서사를 우선 화면 검증한다.
3. 한자 병기 8개는 표시 용어를 확정한 뒤 별도 수동 오버레이로 넣는다.
4. 비문자 토큰 59개는 번역 진척도에 넣지 않고 그대로 둔다.

공개 산출물에는 원문이나 게임 바이너리를 넣지 않는다. 좌표와 해시만 보관한다.
