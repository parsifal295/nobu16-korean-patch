- JP를 기준으로 SC·TC·EN을 대조했다.

- 제어코드, 색상 태그, 동적 이름 토큰, 문자열 종결자, XL13 메타데이터와 원래 컨테이너 래퍼 방식을 보존했다.
- 사용자가 Base DLC와 PK DLC 이벤트가 같은 대화 위젯으로 보인다고 확인한 전제에 따라, 모든 DLC 이벤트에 static patch 007 기준을 적용했다.
  - 런타임 전각 30px
  - 유효 폭 912px 이하
  - 최대 4줄
  - 원시 G1N 폭은 전각 48px·반각 24px로 계산
  - `ceil(raw_width × 30 / 48)`로 유효 폭 환산
  - 동적 이름 토큰은 현재 `MSG_PK/JP/msgev.bin`의 전체 이름 폭을 같은 비율로 예약
