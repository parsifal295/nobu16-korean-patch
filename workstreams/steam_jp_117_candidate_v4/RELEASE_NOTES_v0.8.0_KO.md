# v0.8.0 후보 v4 변경 메모

대상은 Steam PK `1.1.7`(build `18823764`)의 일본어(`JP`) 경로다. 이 후보는
파일 전용 리소스 교체 구조이며 SC 경로, 메모리 패치, DLL 주입, 후킹,
EXE·레지스트리 변경을 사용하지 않는다.

v4 조합에는 다음의 검증된 후속 수정이 포함된다.

- 시나리오 선택의 기간·개월 표기 ID 1051 1건을 보정하고 기간의 개월 값은
  보존한다.
- 이벤트 목록의 가문 표기 159건을 정규화한다.
- 전법 독음 10건을 보정하고, 활성 전법 독음 62개 슬롯에 Latin 문자가 남지
  않았는지 검증한다.
- 관직 목록은 표시명과 독음을 분리한다. 표시명은 한글 관직명 113건,
  독음은 일본어 발음의 한글 표기 8건으로 총 121건이다. 기준 예시는
  `관백/간파쿠`, `우대신/우다이진`, `대납언/다이나곤`,
  `정이대장군/세이이다이쇼군`이다.
- 이벤트 잔여 문장 50건과 분기·조건 표기 16건, 총 66건을 추가 보정한다.

관직의 지리 범주와 막부 ID `16614..16624`는 이번 후보에서 보류한다.
ZIP은 실제 게임 폴더 구조와 파일명을 보존하는 JP 경로 12개만 포함한다.

자산명은 `NOBU16_PK_Korean_Patch_Steam_1.1.7_v0.8.0.zip`이며, 재현 검증
결과는 356,142,112 bytes / SHA-256
`D1C2D91E65D03DC3A57EF7D5D45A4B7381EA7752024062CD91D069F02E2BE354`이다.

추적 가능한 source-free 검증 파일은 `verification.v4.json`이며 SHA-256은
`E900CE49E42CB12B99E1CB4D6E148B2D421AE1D87DCDBA0933571711BB1F69D1`이다.
후보 생성과 검증은 설치 게임 파일을 수정하지 않는다. 실제 적용 및 화면
검증은 별도 승인된 단계에서만 수행한다.
