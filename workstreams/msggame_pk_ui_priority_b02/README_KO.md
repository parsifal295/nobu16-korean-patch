# MSGGAME PK UI 우선 번역 B02

이 배치는 블록 13·14를 맡은 B01과 겹치지 않도록 나머지 블록만 검토한다. 외교 화면 설명과 불가 사유, 감장·별호 선택 안내, 정책 확인창, 전투 시스템 알림, 이벤트 합전 목표, 화면 이동 도움말처럼 실제 UI에 직접 노출될 가능성이 높은 완결 문자열 53개를 번역했다.

각 항목은 간체중문에서 한 레코드의 단일 완결 리터럴이며 일문·영문 레코드 전체 문맥으로 의미를 교차 확인할 수 있는 경우만 채택했다. 동적 값이 문장 바깥 바이트코드로 삽입되는 조각, 이벤트 대사, 인명·호칭처럼 보이는 항목은 제외했다. 좌표 `(7, 2076, 0)`은 독립 버튼이 아니라 동적 문장의 일부이므로 `dynamic_narrative_false_positive`로 명시적으로 제외한다.

빌더는 기존 등록 오버레이와 B01 좌표의 중복, 정확한 번역 대상 카탈로그 소속, 자리표시자·제어 코드·줄바꿈, 괄호 순서, 다국어 화면 폭 예산을 검증한다. 설치된 게임 파일은 쓰지 않고, 오프라인 재구성 결과만 해시로 검증한다. 공개 오버레이·근거·검토 인덱스에는 상용 원문을 넣지 않는다.

```powershell
python workstreams/msggame_pk_ui_priority_b02/build_msggame_pk_ui_priority_b02.py
python -m unittest discover -s workstreams/msggame_pk_ui_priority_b02/tests -v
```
