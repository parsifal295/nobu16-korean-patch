# 일반 NPC 이름·직책 품질 점검 v1

이 작업 묶음은 Steam PC JP 한국어 데이터의 일반 NPC 조합 조각과 고정 NPC 표기를 함께 점검·수정한다.

- 대상: Base `MSG/JP/strdata.bin` block 0, PK `MSG_PK/JP/msgdata.bin`, Base `MSG/JP/ev_strdata.bin`, PK `MSG_PK/JP/msgev.bin`
- 조합 점검: 일반 NPC·직책 39개 조합 경로(중복 앵커 포함). 이 중 사용자 화면에서 확인됐거나 PC EN 조각이 정상 단어로 이어지는 16개 경로만 이번 후보에 반영한다.
- 고정 표기: 비무장 NPC·직책의 확정 오역 10개 좌표군
- 근거: Steam PC의 JP 백업과 PC EN/SC/TC 대조만 사용한다. Switch 번역은 사용하지 않는다.

일본어 조각만 맞고 PC EN 조각이 완성어와 어긋나는 나머지 경로는 실게임 확인 전 보류한다. 생성기는 현재 Steam 기준 해시를 먼저 확인하고, 네 파일 이외에는 출력하지 않는다. 조각 수정 뒤에는 Base와 PK 양쪽에서 같은 조합 결과가 나오는지와 고정 문자열 좌표를 모두 검증한다.

실행 파일과 주요 데이터에서 조각쌍의 직접 런타임 참조는 정적으로 복원되지 않았다. 따라서 `상사람`, `선교모로`, `소성씨`, `덴령`처럼 실제 화면 보고가 있거나 PC EN 조각이 정상 단어로 이어지는 경로만 적용한다. 일본어 완성형만으로 추정되는 조각은 고정 라벨 수정과 분리해 보류한다.

생성 결과는 추적하지 않는 `tmp/npc_name_component_quality_v1/` 아래에만 만든다. 이 작업 묶음은 Steam에 쓰지 않는다.
