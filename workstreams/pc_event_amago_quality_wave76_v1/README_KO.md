# W76 아마고 부흥·시카노스케 결말 이벤트 품질 검수

이 작업 스트림은 PK 이벤트 `7199–7213`의 아마고 부흥·시카노스케 결말 장면을
재검수한다. 입력은 반드시 W75의 **온디스크** 후보
`tmp/pc_event_toyotomi_quality_wave75_v1/candidate-final/MSG_PK/JP/msgev.bin`이며,
W75 재구성과 온디스크 바이트, 압축·원본 프로필이 모두 고정값과 일치하지 않으면
후속 빌드를 중단한다.

수정 행은 정확히 다음 10개다.

`7199, 7200, 7201, 7202, 7203, 7205, 7207, 7211, 7212, 7213`

`7204, 7206, 7208, 7209, 7210`은 직접 PC 일본어 및 PC EN/SC/TC 문맥 대조 후
그대로 유지한다. 일본어 원문의 개행 위치는 이식하지 않는다. 한국어 개행은 의미
단위로만 배치하며, 문장 축약·정보 삭제·인명 변경을 하지 않는다.

각 행은 원본 G1N 전각 48px / 반각 24px로 측정한다. 모든 수동 줄은 최대 4줄,
raw 960px 이하이며, 보고서에는 표시 문자열, raw 폭, `ceil(raw * 30 / 48)` 실효 폭,
전각/반각 수, 초과 여부를 기록한다. 색상 태그·제어 코드·종료 코드·의미 단위는
보존하고 태그 내부에는 개행을 넣지 않는다.
