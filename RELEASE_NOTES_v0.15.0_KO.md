# NOBU16 PK 한글 패치 v0.15.0 릴리즈 노트

## DLC 번역·이미지 아틀라스 검수·성 이름 표시 선택

v0.15.0은 Steam JP PK 1.1.7 순정 설치본에 v0.14.0의 본문 번역,
신규 DLC 번역, 검수한 이미지 아틀라스, 선택한 성 이름 표시 방식을
한 번에 적용하는 직행 통합 패처입니다. v0.14.0을 먼저 설치할 필요는
없습니다.

> **설치 전 필수:** 이전 패치·모드·수동 수정 이력이 있거나 현재 상태가
> 확실하지 않으면 Steam에서 **NOBU16 → 속성 → 설치된 파일 → 게임 파일
> 무결성 확인**을 실행해 Steam JP 1.1.7 순정 상태를 준비하십시오.

## 주요 변경

### 보유한 DLC만 선택 번역

- Base·PK DLC 리소스 **105개**의 번역 프로필을 추가했습니다.
- 105개를 모두 보유할 필요는 없습니다. 패처는 실제로 설치된 파일만
  순정 해시를 확인하고, 없는 DLC는 오류 없이 건너뜁니다.
- 나중에 DLC를 구매·설치했다면 `APPLY_KOREAN_PATCH.bat`를 다시 실행하면
  새로 생긴 순정 DLC 파일만 추가로 번역합니다.
- DLC 통합 후 확인된 화면 제목 4건의 후속 보정도 함께 반영했습니다.

### 이미지 아틀라스 검수본

- 필수 아틀라스를 6개로 확장하고 `BASE LOW`, `PORT1 HIGH`,
  `PORT3 HIGH` 검수본을 반영했습니다.
- PORT3 화면 제목은 영양군 음식디미방체로 통일했습니다. 일본어 원본처럼
  좌측 정렬하고 첫 글자는 크게, 나머지 글자는 작게 조정했으며,
  과한 노란기를 줄이고 원본과 같은 부드러운 테두리를 반영했습니다.
- PORT3와 대응하는 저해상도 제목은 `BASE LOW`에도 같은 표현으로
  반영했습니다.
- PORT1 HIGH에서 누락 확인된 버튼 이미지도 검수 시안으로 교체했습니다.

### 성 이름 세로쓰기·가로쓰기 선택

`APPLY_KOREAN_PATCH.bat`를 실행하면 두 가지 EXE 프로필 중 하나를
선택합니다.

- `1` 세로쓰기: `001`, `002`, `003`, `007`, `009`, `010` 적용
  - 가로 라벨과 그에 연동된 지도 아이콘 이동 패치 `004`, `005`, `006`,
    `008`은 적용하지 않습니다.
- `2` 가로쓰기: `001~010` 전체 적용

통합 패처를 다시 실행해 다른 번호를 고르면 프로필을 전환합니다.
매번 보호된 순정 EXE 백업에서 선택한 프로필을 다시 만들기 때문에,
이전 표시 방식의 잔여 패치가 남지 않습니다.

## 설치 및 복원

1. Steam에서 게임 언어를 **Japanese**로 지정합니다.
2. Steam 게임 파일 무결성 확인을 완료합니다.
3. Steam과 `NOBU16PK.exe`를 완전히 종료합니다.
4. ZIP의 모든 파일을 `...\SteamLibrary\steamapps\common\NOBU16` 최상위에 풉니다.
5. `APPLY_KOREAN_PATCH.bat`를 실행하고 `1` 또는 `2`를 선택합니다.
6. `패치 완료! Steam에서 게임을 시작하세요.` 문구를 확인한 뒤 Steam에서
   게임을 시작합니다.

전체를 순정 상태로 되돌리려면 게임과 런처를 종료한 뒤
`RESTORE_KOREAN_PATCH.bat`를 실행합니다. 적용 중 순정 해시가 아닌 파일은
안전을 위해 수정하지 않으며, 실패했다면 파일을 수동으로 덮어쓰지 말고
Steam 무결성 확인 후 다시 시도하십시오.

## 패키지 구성과 범위

- 필수 리소스 16개: 본문 텍스트 10개 + 이미지 아틀라스 6개
- 선택 DLC 리소스 105개
- 텍스트 10개는 고정 작업 원장으로 재구성
- 이미지 6개와 DLC 105개는 원본 검증형 BSDIFF40 델타로 적용
- 완전한 게임 리소스와 `NOBU16PK.exe`는 ZIP에 포함하지 않음
- 진행 중인 `workstreams/pc_dialogue_full_retranslation_v0150` 전체 대사
  재번역은 완료 범위가 아니므로 이번 배포에 포함하지 않음
