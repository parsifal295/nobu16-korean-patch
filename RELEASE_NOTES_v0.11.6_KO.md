## v0.11.6 — NPC 표기·인물 대사·이벤트 줄바꿈·상단 헤더 누적 보정

### 수정

- v0.11.5와 비교해 게임 텍스트 리소스 **6개**가 바뀌었고, 실제 변경
  레코드·슬롯은 **276개**입니다. 6은 파일 수이지 수정 항목 수가 아닙니다.
- Base/PK 인물 대사 **205개**, 일반 NPC 조합 조각 **46개**, 이벤트 정적
  표기·줄바꿈 **25개**를 보정했습니다.
- NPC 표기는 누적 반영분의 `덴령` → `전령`, `상사람` → `상인`, `선교모로` → `선교사`,
  `소성씨` → `시동`, `가인` → `가신`을 유지합니다. 이번 전수 점검에서는
  `우에무라 요리카도` → `우에무라 라이렌`, `후쿠시마 마사노부` → `쿠시마 마사노부` 등 고정 인명 6건과 일반 NPC 조합 23개를 추가로
  바로잡았습니다.
- 정적 대사 20쌍은 Base/PK 양쪽에 함께 반영했습니다. 대상 대사는 실제 활성
  글꼴에서 모두 3줄 이하·912px 이하가 되도록 검증했습니다.
- 정적 설치기는 기존 무장 에디트·가공 히메 작명 입력 보정 9개에 상단 헤더
  가로 배치 12개를 더해 총 21개 지점을 갱신합니다. `NOBU16PK.exe`는 ZIP에
  넣지 않습니다. 001·002·003 데이터 파일을 읽는 마스터 설치기가 각 항목의
  적용 상태를 판정하고 Pending 항목만 적용합니다. 부분 적용 상태는 EXE를
  교체하지 않고 중단하며, 별도 레지스트리가 패치 파일과 최종 해시를 고정합니다.
- v0.11.4의 정책 효과 `%` 표기 88개(공통 39개·PK 49개), 이미지·폰트 자산과
  나머지 게임 리소스 9개는 변경하지 않았습니다.

### 검증

- 순정 PC JP와 PC EN/SC/TC만으로 대상·문맥을 대조했습니다. 스위치판 한글은
  읽거나 기준으로 사용하지 않았습니다.
- NPC 후보 58슬롯, 정적 대사 후보 40레코드, 두 후보의 11파일 결합 프로필을
  각각 회귀 검증했습니다. 비대상 NPC 테이블 레코드는 바이트 동일하며, 정책
  효과의 `%` 리터럴과 printf 토큰도 바이트 동일합니다.
- Steam JP 11개 텍스트 파일을 대상으로 백업 생성, 사전 드라이런, 적용,
  사후 드라이런을 모두 통과했습니다. 최종 적용본은 후보 해시와 일치합니다.
- 상단 헤더 설치기 12개 지점은 2048×1152에서 게임을 완전히 종료한 뒤
  재실행해 확인한 값이며, 이전 정적 설치본에서도 업그레이드할 수 있습니다.
- 배포 ZIP은 게임 리소스 15개와 설치·지원 파일 15개, 총 30개로 구성됩니다.
  `NOBU16PK.exe`는 포함하지 않습니다.

### 설치

1. Steam/런처의 게임 언어를 **Japanese**로 설정합니다.
2. 게임과 런처를 완전히 종료합니다.
3. ZIP 전체를 게임 폴더에 덮어씁니다.
4. 정적 EXE 보정을 사용할 경우 `APPLY_STATIC_EXE_PATCHES.bat`를 한 번
   실행합니다. 기존 `APPLY_STATIC_OFFICER_EDITOR_FIX.bat`도 같은 마스터를
   호출하는 호환 진입점입니다.
