# 신장의 야망·신생 with 파워업키트 비공식 한글 패치

Windows PC판 `NOBU16PK.exe`를 기준으로 제작 중인 파일 전용 한글 패치입니다.
공식 간체중문(SC) 가로쓰기 리소스 경로를 사용하며, 메모리 패치·DLL 주입·후킹·
EXE 수정·레지스트리 변경은 사용하지 않습니다.

> 현재 상태: **개발 중**
>
> 현재 적용 파일을 묶은 최신 개발 배포본은
> [GitHub Releases](https://github.com/parsifal295/nobu16-korean-patch/releases)에서 받을 수 있습니다.
> 번역과 화면 검수는 아직 진행 중입니다.

## 적용 전 필수 설정

공식 게임 런처의 언어 설정에서 반드시 **`简体中文 / Simplified Chinese`**를 선택해야
한글 패치가 적용됩니다. 이 패치는 `MSG/SC`, `MSG_PK/SC`, `RES_SC` 리소스를 사용하므로
일본어·영어·번체중문을 선택하면 한글 파일이 로드되지 않습니다.

1. 공식 런처에서 `简体中文 / Simplified Chinese`를 선택합니다.
2. 게임과 런처를 완전히 종료합니다.
3. 배포 ZIP의 폴더를 게임 설치 폴더에 덮어씁니다.
4. 공식 런처로 게임을 다시 실행합니다.

## 현재 포함 범위

- PK 전용 UI, 이벤트 대사, 장수·성·지역 데이터, 장수 열전, 물품 설명, 크레디트
- PK 실행 중 함께 읽는 공용 `MSG/SC/strdata.bin`
- 서울한강체 기반 한글 글리프 후보와 일부 화면 제목 이미지

본편 전용 `MSG/SC/ev_strdata.bin` 등 PK 실행에서 사용이 확인되지 않은 파일은 진행률과
적용 대상에서 제외합니다.

## 아직 남은 작업

- `msggame`과 `msgdata`의 남은 문구 번역
- 실제 게임 화면에서 글꼴 굵기, 누락 글리프, 줄바꿈과 잘림 검수
- 저장·불러오기·종료 확인과 완전 복원 가능한 설치본 검증
