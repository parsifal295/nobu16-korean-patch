# 노부나가의 야망 신생 PK 한글 패치

Windows Steam판 `NOBU16PK.exe` 일본어(JP) 리소스 경로를 사용하는 비공식
팬메이드 한글 패치입니다. 대상은 Steam JP PK v1.1.7(`Update 1.1.7`, Steam
build `18823764`)이며, Steam/런처의 게임 언어도 반드시 **Japanese**로
선택해야 합니다.

현재 공개 안정판은 [v0.15.1](https://github.com/parsifal295/nobu16-korean-patch/releases/tag/v0.15.1)입니다.
v0.15.1은 **순정 Steam JP 1.1.7 설치본에서 바로 적용하는 직행 통합 패처**입니다.
`APPLY_KOREAN_PATCH.bat` 한 번으로 정적 EXE, 필수 리소스 16개, 실제로
설치된 DLC 번역을 함께 적용합니다. 설치 중 성 이름 표시를
`1` 세로쓰기 또는 `2` 가로쓰기로 선택할 수 있고, 보유하지 않은
DLC 파일은 오류 없이 건너뜁니다. 완전 게임 리소스와 `NOBU16PK.exe`는
ZIP에 포함하지 않습니다.

## v0.15.1 — PORT3 순정 프로필 핫픽스

- v0.15.0에서 `RES_JP_PK_PORT/res_lang_pk_port3.bin`의 작업 중간본을
  순정 원본으로 잘못 판정해, 실제 순정 설치본이 `Resource stage Preflight`에서
  중단되던 문제를 수정했습니다.
- 올바른 순정 PORT3 SHA-256은
  `BE1361E17341D433931EB5740B228EF1842BF6DF2F01D4F582CE790A9A57A154`입니다.
- v0.15.1의 번역·DLC·이미지·성이름 표시 기능 범위와 최종 목표 파일은
  v0.15.0과 동일하며, PORT3 입력 프로필과 바이너리 델타만 바로잡았습니다.
- v0.15.0 설치가 `writes_performed: false`로 중단됐다면 아무 파일도 적용되지
  않은 상태입니다. Steam 무결성 확인 후 v0.15.1을 설치하십시오.

### DLC 번역·이미지 검수·성 이름 표시 선택

### 설치 전 필수 사항

> v0.15.1의 첫 설치는 **Steam JP 1.1.7 순정 파일**에서 시작합니다.
> 이전 한글 패치·모드·수동 수정 이력이 있거나 상태가 확실하지 않으면
> Steam 라이브러리에서 **NOBU16 → 속성 → 설치된 파일 → 게임 파일 무결성 확인**을
> 실행해 원본 상태를 준비하십시오.

1. Steam/런처에서 게임 언어를 **Japanese**로 설정합니다.
2. 게임 파일 무결성 확인을 완료한 뒤 Steam과 `NOBU16PK.exe`를 완전히
   종료합니다.
3. v0.15.1 ZIP의 **모든 파일**을
   `...\SteamLibrary\steamapps\common\NOBU16` 최상위에 풉니다.
4. `APPLY_KOREAN_PATCH.bat`를 실행합니다.
5. 프롬프트에서 성 이름 표시 방식을 선택합니다.
   - `1`: 세로쓰기. 가로 라벨과 관련 지도 아이콘 이동 패치 `004`, `005`,
     `006`, `008`을 제외합니다.
   - `2`: 가로쓰기. 가로 라벨과 관련 지도 아이콘 이동 패치를 모두
     적용합니다.
6. `패치 완료! Steam에서 게임을 시작하세요.`가 표시되면 Steam에서
   게임을 시작합니다.

순정 상태로 되돌리려면 게임과 런처를 종료한 뒤
`RESTORE_KOREAN_PATCH.bat`를 실행합니다. 표시 방식만 바꾸려면 통합 패처를
다시 실행하고 다른 번호를 선택하면 됩니다.

### DLC 선택 설치

- 번역 프로필은 Base·PK DLC 리소스 **105개**를 지원합니다.
- 사용자가 실제로 보유하고 설치한 DLC 파일만 순정 해시를 확인한 뒤
  번역합니다. 없는 파일은 정상적으로 건너뜁니다.
- 패치 설치 후 DLC를 새로 추가했다면 `APPLY_KOREAN_PATCH.bat`를 다시
  실행하십시오. 이미 적용된 필수 파일은 유지하고, 새로 생긴 순정
  DLC 파일만 추가로 번역합니다.

### 이미지 아틀라스 검수

- 필수 이미지 아틀라스를 6개로 확장하고 `BASE LOW`, `PORT1 HIGH`,
  `PORT3 HIGH` 검수본을 반영했습니다.
- 화면 제목은 영양군 음식디미방체를 기준으로 통일했고, 원본의
  좌측 정렬·첫 글자 강조·부드러운 테두리 표현을 반영했습니다.
- 해상도를 변경했다면 이미지 적용 여부를 판단하기 전에
  `NOBU16PK.exe`를 완전히 종료하고 다시 실행하십시오.

### 패키지·검증

- 필수 리소스는 본문 텍스트 10개와 이미지 아틀라스 6개, 총 16개입니다.

## v0.14.0 — 번역·표기 보정·이벤트 메시지 가독성·지도 보조 표시 정렬

### 순정 Steam JP 직행 통합 패처

> **설치 전 필수 — Steam 원본 상태**
>
> v0.14.0의 신규 설치는 **Steam JP 1.1.7 순정 파일에서만 가능**합니다.
> 이전 한글 패치, 모드, 수동 파일 수정이 남아 있거나 상태가 확실하지 않다면
> Steam 라이브러리에서 **NOBU16 → 속성 → 설치된 파일 → 게임 파일 무결성 확인**을 실행해
> 원본 상태로 복구한 뒤 설치하십시오. 일부 파일만 바뀐 설치본에서는 안전을 위해 패처가 중단됩니다.

- v0.14.0은 **정확한 순정 Steam JP 1.1.7 EXE와 15개 리소스 프로필**에서
  시작합니다. v0.13.x를 먼저 설치할 필요가 없으며, 이미 수정된 리소스나 일부
  파일만 바꾼 설치본은 안전을 위해 중단합니다.
- 패키지에는 완전한 게임 리소스와 `NOBU16PK.exe`가 들어 있지 않습니다.
  적용 전 15개 리소스와 정적 EXE 상태를 모두 검사하고, 텍스트 10개는 고정
  레저로 재구성하며 폰트·이미지 5개는 원본 검증형 BSDIFF40 델타로 복원합니다.
- 적용 전의 15개 파일은 게임 폴더 아래
  `KR_PATCH_BACKUP/v0.14.0-direct-patcher`에 보관합니다.
  원본 EXE는 `NOBU16PK.exe.staticfix.original_1.1.7`에 보관합니다.
- 통합 적용은 정적 EXE → 리소스 순서, 복원은 리소스 → 정적 EXE 순서로 처리합니다.
  리소스 단계가 실패하면 이번 실행에서 순정 EXE를 바꾼 경우 EXE도 자동으로
  원복합니다.
- 정적 EXE 처리는 게임 실행 중 메모리를 건드리는 helper가 아닙니다. 원본을
  로컬에서 검증해 한 번 교체하는 방식이며, 매 게임 세션마다 별도 실행 파일을 실행할 필요가 없습니다.

### 설치 방법

1. Steam에서 NOBU16의 게임 언어를 **Japanese**로 설정합니다.
2. Steam 라이브러리에서 **NOBU16 → 속성 → 설치된 파일 → 게임 파일 무결성 확인**을
   실행하고 완료될 때까지 기다립니다. 이 단계로 Steam JP 1.1.7 원본 상태를 준비해야 합니다.
3. Steam과 `NOBU16PK.exe`를 완전히 종료합니다.
4. v0.14.0 ZIP의 **모든 파일**을 게임 폴더 최상위에 풉니다. 일반적인 경로는
   `...\SteamLibrary\steamapps\common\NOBU16`입니다.
5. `APPLY_KOREAN_PATCH.bat`를 한 번 실행합니다. 사전 검사를 통과하면 정적 EXE 패치
   `001~010`과 15개 한글 리소스를 함께 적용합니다.
6. `패치 완료! Steam에서 게임을 시작하세요.`가 표시되면 Steam에서 게임을 실행합니다.

적용이 중단되면 파일을 수동으로 덮어쓰지 말고, Steam 무결성 확인으로 원본 상태를
복구한 뒤 다시 실행하십시오. 전체를 순정 상태로 되돌리려면 게임과 런처를 종료한 뒤
`RESTORE_KOREAN_PATCH.bat`를 실행합니다.

### 번역·표기 변경

- Base·PK 인물 대사 B17의 **17개 레코드, 24개 리터럴**을 원문 구조와 대조해
  이름 경계 공백, 조사, 부자연스러운 문장 연결을 보정했습니다.
- PK 이벤트 대사 `MSG_PK/JP/msgev.bin`은 아네가와 전투 14건을 포함한 **20건**의
  수동 줄바꿈을 문맥 단위로 다시 배치했습니다. 문장을 축약하지 않고 색상 태그와
  런타임 토큰을 보존했으며, 각 줄은 30px·유효 폭 912px·최대 4줄 기준을 통과했습니다.
- 무장 성·이름 독음 칸은 한국 한자음으로 전환했습니다. PK `msgdata.bin` 4,569칸과
  Base `strdata.bin` 4,556칸이 대상이며, 예를 들어 `직전신장`처럼 공백 없는 한글
  독음만 사용합니다. 일반 표시 무장명은 바꾸지 않습니다.
- 동적 조합 성명이 붙어 보이던 전용 별칭 성 블록 5건은 성·이름 사이 공백을
  보정했습니다. 성·이름 양쪽에 재사용되는 조각은 범위에서 제외했습니다.

### 정적 변경

- 정적 설치기 패치 `007-EventMessageTypography`를 추가해 이벤트 메시지의
  글자 크기를 36에서 30으로, 줄간격을 10에서 8로 조정합니다.
- 이벤트 메시지 생성 루틴의 두 상수만 변경합니다. 일반 UI, 무장 목록,
  지도 라벨 등 다른 텍스트 위젯의 글꼴 설정에는 영향을 주지 않습니다.
- 정적 설치기 패치 `009-EventMessageParentWidth`는 이벤트 내레이션 부모 폭을
  882에서 972로 넓혀, 좌측 60px 여백 뒤에 912px 텍스트 레인을 확보합니다.
- 정적 설치기 패치 `010-EventMessageAutoWrapLimit`는 실제 이벤트 포매터의
  주·대체 두 자동줄바꿈 경로 모두에서 한도를 전각 20자급(40 단위)에서
  전각 30자급(60 단위)으로 넓힙니다.
  번역 데이터에 작성한 수동 개행과 제어 태그는 그대로 보존합니다.
- 정적 설치기 패치 `008-HorizontalMapAuxiliaryIndicators`를 추가해 임전 숫자를
  이동된 상태 아이콘 바로 오른쪽에 붙이고, 성주가 임명되지 않은 성의 빨간
  느낌표를 실제 성 이름 라벨의 오른쪽 끝에 배치합니다.
- 임전 숫자 X는 원래 임전 아이콘과 숫자의 상대 간격 `50-30=20px`를 보존해
  이동된 상태 아이콘 X에 더하고, Y는 두 위젯의 실제 높이로 하단 정렬합니다.
  성주 미임명 느낌표 Y는 일반 높이 필드가 아니라 전용
  32px 스프라이트 높이를 사용해 40px 성 이름 라벨 안에 중앙 정렬합니다.
- 성 문장과 그 아래의 일반 병사수 표시는 008에서도 변경하지 않습니다.

## v0.13.1 — 지도 성 이름 뒤 상태 아이콘 동적 정렬

- Issue #72에서 가로쓰기 성 이름과 전투 준비·방위 거점·공략 목표 아이콘이
  서로 떨어져 보이던 문제를 수정합니다.
- 상태 아이콘 X는 고정값이 아니라 현재 렌더링된 성 이름 라벨 폭을 더해
  계산합니다. Y도 라벨과 아이콘의 실제 높이로 중앙 정렬하므로 이름 길이나
  아이콘 종류가 달라져도 같은 기준으로 배치됩니다.
- 성 문장 아이콘과 그 바로 아래의 병사수(병종 아이콘+숫자) 표시 그룹은
  엔진의 원래 좌표를 유지합니다. 특수 지도 커서 경로도 변경하지 않습니다.
- 정적 설치기 레지스트리에 같은 크기의 `006` BytePatch를 추가합니다.
  v0.13.0 설치본에서는 `006`만, v0.12.0 설치본에서는 `005~006`만 적용하며,
  원본에서는 `001~006`을 순서대로 적용합니다.
- 1920×1080에서 `NOBU16PK.exe`를 완전히 종료한 뒤 새 프로세스로 재실행해
  나고야성 방위 거점, 공략 목표, 전투 준비의 동적 정렬과 병사수 표시가 성
  문장 바로 아래에 유지되는 것을 확인했습니다.


1.1.7 런처에서 시작하십시오. 설치 도구는 Windows .NET Framework 4.5.2
이상이 필요하며, 포함된 Steamless v3.1.0.5 구성요소는 atom0s의 원본 그대로

`OfficerEditorStaticFix\THIRD_PARTY_NOTICES.txt`에 있습니다.

`v0.10.0`은 Steam JP 1.1.7용 공개 배포본입니다. 게임 내 번역 대상 텍스트 번역을 완료했습니다.
현재는 순정 PC 일본어 원문과 PC EN/SC/TC 대조로 인물·이벤트 대사 번역 품질을 전수 감사하고 있으며, 스위치판 한글은 기준으로 사용하지 않습니다.

공식 크레딧·일본어 IME 등 번역 대상이 아닌 구조용 슬롯은 유지하고, 고확신 오역만 별도 검증 후 수정합니다.
이벤트 스크립트 줄바꿈 검수·보정과 이미지 번역은 진행 중입니다.
현재 정적 후보인 이벤트 26건과 인물 대사 51건은 PC 원문 기준으로 합성·구조 검증한 뒤 개발 Steam JP 설치본에 적용해 부팅·저장 불러오기까지 확인했습니다.
이 검증 묶음은 v0.13.0 번역 품질 개선 범위에 포함하며, 실제 대사·이벤트 화면 QA와 후속 런타임 항목 검수는 계속 진행 중입니다.

## 배포 및 기여 정책

본 한글 패치의 수정본 제작·배포와 전체 파일의 재배포는 허용하지 않습니다.
공식 배포본은 이 저장소의
[GitHub Releases](https://github.com/parsifal295/nobu16-korean-patch/releases)에서만
제공합니다.

오류 수정, 번역 개선, 누락 또는 오표기 제안은 공식 저장소의
[Issues](https://github.com/parsifal295/nobu16-korean-patch/issues) 또는
[Pull Requests](https://github.com/parsifal295/nobu16-korean-patch/pulls)를 이용해
제출해 주세요. 제안된 변경은 검토 후 공식 배포본에 반영됩니다.

공식 배포처 외의 경로에서 받은 파일은 최신성, 무결성, 호환성 및 정상 동작을
보장하지 않습니다.

## 글꼴과 권리

한글 글리프는 서울한강체 계열을 사용하며, 32px/4px 계열은 서울한강 B,
48px/6px 계열은 서울한강 EB로 렌더링했습니다.

이 프로젝트는 비공식 팬메이드 한글 패치입니다. KOEI TECMO GAMES의 공식
제품·지원물이 아니며, 게임·상표·로고·원본 데이터의 모든 권리는 KOEI
TECMO GAMES 및 각 권리자에게 있습니다. `NOBU16PK.exe` 게임 실행 파일은
배포 ZIP에 포함하지 않습니다.
