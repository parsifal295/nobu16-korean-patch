# 신장의 야망·신생 with 파워업키트 비공식 한글 패치

Windows PC판 `NOBU16PK.exe`용 **일본어판 리소스 경로 기반** 파일 전용 한글 패치입니다.
메모리 패치, DLL 주입, 후킹, EXE·레지스트리 변조는 사용하지 않습니다.

> 현재 상태: **개발 중**
>
> 일본어 10파일 조합은 로컬 비Steam PC에서 실제 실행·한글 글리프 화면 확인을
> 통과했습니다. Steam판의 파일 조합 A/B와 화면 QA는 아직 끝나지 않았습니다.

## 적용 전 필수 설정

Steam 라이브러리의 게임 언어와 공식 런처 언어를 모두 **`日本語 / Japanese`**로
설정해야 합니다. 게임과 런처를 완전히 종료한 뒤 패치 폴더를 게임 설치 폴더에
그대로 덮어씁니다.

일판 전환 후보는 다음 **10개 파일**을 원래 이름과 폴더 구조 그대로 사용합니다.

- `MSG/JP/strdata.bin` 1개
- `MSG_PK/JP/` 아래 `msgui.bin`, `msgev.bin`, `msgdata.bin`, `msgbre.bin`,
  `msgire.bin`, `msgstf.bin`, `msggame.bin` 7개
- `RES_JP/res_lang.bin` 1개
- `RES_JP_PK/res_lang_pk.bin` 1개

따라서 ZIP 최상단에는 `MSG`, `MSG_PK`, `RES_JP`, `RES_JP_PK` 네 폴더가 있어야
합니다. 현재 후보는 정적 분석에서 별도 글꼴 컨테이너가 확인된 `RES_JP`와
`RES_JP_PK`를 모두 포함합니다. 실제 Steam PK 런타임의 필수 파일 조합은 화면 A/B
QA로 확정합니다.
